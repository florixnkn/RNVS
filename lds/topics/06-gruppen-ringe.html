Worum geht's?

Eine Gruppe ist eine Menge mit einer Verknüpfung, die sich "wie ein vernünftiges Spiel" verhält: Du kannst Züge kombinieren, es gibt einen "Nichts-tun"-Zug, und jeder Zug lässt sich rückgängig machen. Diese vier Eigenschaften bilden die meistgeprüfte Checkliste der ganzen Klausur — sie taucht praktisch jedes Semester auf.

Stell dir vor …

… einen Zauberwürfel. Jede Drehung ist ein "Element" deiner Menge, "Hintereinander-Drehen" ist die Verknüpfung ⊙. Abgeschlossenheit: Nach jeder Drehfolge hast du immer noch einen gültigen Würfelzustand. Assoziativität: Egal wie du Klammern setzt — (Dreh1 dann Dreh2) dann Dreh3 = Dreh1 dann (Dreh2 dann Dreh3). Neutrales Element: "Gar nicht drehen" ändert nichts. Inverses Element: Jede Drehung lässt sich durch die Gegendrehung rückgängig machen. Erfüllt eine Struktur alle vier Punkte, ist sie eine Gruppe.

Gruppenaxiome

Definition: (G, ⊙) ist eine Gruppe, wenn vier Bedingungen gelten:

#AxiomFormal
G1Abgeschlossenheita ⊙ b ∈ G für alle a,b ∈ G
G2Assoziativität(a ⊙ b) ⊙ c = a ⊙ (b ⊙ c)
G3Neutrales Element∃e ∈ G: e ⊙ a = a ⊙ e = a für alle a
G4Inverse Elemente∀a ∃a⁻¹: a ⊙ a⁻¹ = a⁻¹ ⊙ a = e

Gilt zusätzlich a ⊙ b = b ⊙ a für alle a,b, heißt die Gruppe abelsch (kommutativ).

Beispiel: (ℤ₅*, ·) — Gruppe prüfen

ℤ₅* = {1,2,3,4} mit Multiplikation mod 5. Verknüpfungstabelle:

·1234
11234
22413
33142
44321
  1. Abgeschlossenheit: Alle Einträge in {1,2,3,4} ✓
  2. Assoziativität: Multiplikation ist immer assoziativ ✓
  3. Neutrales Element: Zeile 1 = Spalte 1 = Identität → e = 1 ✓
  4. Inverse: 1⁻¹=1, 2⁻¹=3, 3⁻¹=2, 4⁻¹=4 (jede Zeile enthält eine 1) ✓
Ergebnis

(ℤ₅*, ·) erfüllt alle vier Axiome → ist eine Gruppe. Da die Tabelle symmetrisch zur Diagonale ist, ist sie sogar abelsch.

Untergruppen

Definition: H ⊆ G ist Untergruppe, wenn (H, ⊙) selbst eine Gruppe ist.

Untergruppen-Test (Kurzform)
H ≠ ∅ und ∀a,b ∈ H: a ⊙ b⁻¹ ∈ H

Äquivalent und oft einfacher zu prüfen: H ≠ ∅, H ist abgeschlossen unter ⊙, und jedes Element hat sein Inverses in H.

Beispiel: Untergruppen von (ℤ₅*, ·) = {1,2,3,4}

KandidatTestErgebnis
{1}trivial, alle Bedingungen erfüllt✅ Untergruppe
{1,2}2·2 = 4 ∉ {1,2}❌ keine Untergruppe
{1,3}3·3 = 4 ∉ {1,3}❌ keine Untergruppe
{1,4}4·4 = 1 ∈ {1,4} ✓✅ Untergruppe
{1,2,3,4}ganze Gruppe✅ Untergruppe (unecht)

Ringe & Körper

Definition: (R, +, ·) ist ein Ring, wenn:

  1. (R, +) ist abelsche Gruppe (mit Nullelement 0)
  2. (R, ·) ist Halbgruppe (nur Abgeschlossenheit + Assoziativität — keine Inversen nötig!)
  3. Distributivgesetze: a·(b+c) = a·b + a·c (und (a+b)·c = a·c + b·c)
Ring(R,+) Gruppe + (R,·) Halbgruppe + Distributivität
KörperRing + (R\{0},·) ist abelsche Gruppe
Beispiel Ring(ℤ, +, ·) — kein Körper, da 2⁻¹ ∉ ℤ
Beispiel Körper(ℚ, +, ·), (ℝ, +, ·), (ℤ₅, +, ·)

🎓 Klausuraufgabe: Struktur-Checkliste

Klausur SoSe 2025, Aufgabe 4 (12 Punkte) — die klassische Checklisten-Aufgabe. Gegeben: (ℤ×ℤ, ⋆) mit (a,b) ⋆ (c,d) = (a+c, b+d+a·c). Prüfe: Halbgruppe? Monoid? Gruppe? Kommutativ?

Sanity-Check mit konkreten Zahlen

(1,2) ⋆ (3,4) = (1+3, 2+4+1·3) = (4, 9) — so ein Zahlenbeispiel hilft, die Formel vor dem allgemeinen Beweis zu verstehen.

  1. Halbgruppe? (Assoziativität prüfen)
    Linke Seite: [(a,b)⋆(c,d)]⋆(e,f) = (a+c, b+d+ac) ⋆ (e,f) = (a+c+e, b+d+f+ac+ae+ce)
    Rechte Seite: (a,b)⋆[(c,d)⋆(e,f)] = (a,b) ⋆ (c+e, d+f+ce) = (a+c+e, b+d+f+ce+ac+ae)
    Beide Ausdrücke sind identisch (nur die Summanden anders sortiert) → assoziativ ✓. Abgeschlossenheit ist offensichtlich (Ergebnis wieder in ℤ×ℤ). → Halbgruppe: JA
  2. Monoid? (neutrales Element finden)
    Test e=(0,0): (0,0)⋆(a,b) = (0+a, 0+b+0·a) = (a,b) ✓
    (a,b)⋆(0,0) = (a+0, b+0+a·0) = (a,b) ✓
    → Monoid: JA, neutrales Element ist (0,0).
  3. Gruppe? (Inverses finden)
    Gesucht (x,y) mit (a,b)⋆(x,y) = (0,0): (a+x, b+y+ax) = (0,0)
    Aus a+x=0 folgt x = −a. Eingesetzt: b+y+a(−a) = 0 ⟹ y = −b+a².
    Probe (andere Richtung): (x,y)⋆(a,b) = (−a+a, (−b+a²)+b+(−a)·a) = (0, −b+a²+b−a²) = (0,0) ✓
    → Gruppe: JA, Inverses: (a,b)⁻¹ = (−a, −b+a²)
  4. Kommutativ?
    (a,b)⋆(c,d) = (a+c, b+d+ac)  vs.  (c,d)⋆(a,b) = (c+a, d+b+ca)
    Da + und · in ℤ kommutativ sind (a+c=c+a, ac=ca), sind beide Ergebnisse gleich → kommutativ: JA
Ergebnis

(ℤ×ℤ, ⋆) ist eine abelsche Gruppe mit neutralem Element (0,0) und Inversem (a,b)⁻¹ = (−a, −b+a²). Diese vier Punkte (Halbgruppe → Monoid → Gruppe → kommutativ) sind exakt die Reihenfolge, in der du jede „algebraische Struktur"-Aufgabe abarbeiten solltest.

Typische Fallen

❌ Falsch"Die Verknüpfung sieht aus wie +/·, also automatisch assoziativ."
✅ RichtigBei custom Verknüpfungen (wie a⋆b = a+b+ab) MUSS Assoziativität explizit nachgerechnet werden — nichts wird automatisch angenommen.
❌ FalschNeutrales Element nur in einer Richtung testen: e⊙a = a reicht.
✅ RichtigEs MUSS beide Richtungen gelten: e⊙a = a UND a⊙e = a. Bei nicht-kommutativen Verknüpfungen reicht eine Richtung nicht.
❌ FalschInverses nur für "einfache" Elemente prüfen, oder nur a⊙b=e zeigen.
✅ RichtigFür jedes a ∈ G muss ein a⁻¹ existieren, und zwar mit a⊙a⁻¹ = a⁻¹⊙a = e (beide Richtungen).
❌ Falsch∅ ist eine Untergruppe.
✅ RichtigJede Untergruppe muss mindestens das neutrale Element e enthalten — H ≠ ∅ ist Pflichtbedingung.
❌ Falsch"(R,+) Gruppe und (R,·) Halbgruppe reichen für einen Ring."
✅ RichtigZusätzlich MUSS die Distributivität a·(b+c) = a·b+a·c gelten — sie verbindet die beiden Verknüpfungen.

Quick-Reference

🎯 Quick-Reference für die Klausur

Gruppen-Checkliste

Gegeben: (G, ⊙)

☐ Abgeschlossenheit: a⊙b ∈ G für alle a,b ∈ G?
☐ Assoziativität: (a⊙b)⊙c = a⊙(b⊙c) für alle?
☐ Neutrales Element: ∃e mit e⊙a = a⊙e = a?
☐ Inverse Elemente: ∀a ∃a⁻¹ mit a⊙a⁻¹ = a⁻¹⊙a = e?

Falls alle JA: GRUPPE ✓ — Falls ein NEIN: KEINE GRUPPE ✗
Bonus: kommutativ (abelsch)? a⊙b = b⊙a für alle?

Untergruppen-Test

H ⊆ G ist Untergruppe ⟺
☐ H ≠ ∅ (enthält mindestens e)
☐ ∀a,b ∈ H: a⊙b⁻¹ ∈ H

äquivalent: H ≠ ∅, abgeschlossen, und alle Inverse in H

Ring- & Körper-Checkliste

Gegeben: (R, +, ·)

☐ (R,+) ist abelsche Gruppe?
☐ (R,·) ist Halbgruppe? (Abgeschlossenheit + Assoziativität)
☐ Distributivität: a·(b+c) = a·b + a·c?

Falls alle: RING ✓
Körper: Ring + (R\{0},·) ist abelsche Gruppe

EigenschaftErgebnis

Monoid✅ Ja, neutral = (0,0)

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.Ist (ℕ, +) eine Gruppe?

(ℕ, +) ist nur ein Monoid: Abgeschlossenheit, Assoziativität und neutrales Element (0) gelten, aber es gibt kein −a ∈ ℕ. Keine Gruppe!

2.Wie lautet 3⁻¹ in (ℤ₅*, ·)? (Also: welche Zahl x erfüllt 3·x ≡ 1 mod 5?)

3·2 = 6 ≡ 1 (mod 5), also 3⁻¹ = 2.

3.Nenne die einzige echte, nicht-triviale Untergruppe von (ℤ₅*, ·) = {1,2,3,4}.

{1,4} ist abgeschlossen (4·4=16≡1) und enthält alle nötigen Inversen — die einzige echte, nicht-triviale Untergruppe ist {1,4}.

4.🎓 Klausur 2025: Ist (ℤ×ℤ, ⋆) mit (a,b)⋆(c,d) = (a+c, b+d+a·c) eine abelsche Gruppe?

Assoziativ, neutrales Element (0,0), jedes (a,b) hat Inverses (−a,−b+a²), und da + und · in ℤ kommutativ sind, ist ⋆ auch kommutativ → abelsche Gruppe.

5.🎓 Klausur 2025: Das Inverse von (a,b) bzgl. ⋆ ist (−a, −b+a²). Berechne die zweite Komponente des Inversen von (3, 2).

−b+a² = −2+3² = −2+9 = 7. Das Inverse von (3,2) ist also (−3, 7).

6.Was fehlt einem Ring (R,+,·), damit er ein Körper ist?

Ein Ring hat bei · nur eine Halbgruppe. Ein Körper verlangt zusätzlich, dass alle Elemente außer 0 ein multiplikatives Inverses haben — also (R\{0},·) ist abelsche Gruppe.
