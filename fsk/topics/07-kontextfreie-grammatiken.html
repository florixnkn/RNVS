Worum geht's?

Ein endlicher Automat (Thema 02) hat ein Problem: Er kann sich keine beliebig große Zahl merken. Deshalb kann kein DFA/NFA erkennen, ob Klammern korrekt verschachtelt sind oder ob genau gleich viele a's wie b's vorkommen. Eine kontextfreie Grammatik (CFG) löst genau das: Sie ist eine Bauanleitung mit Ersetzungsregeln, die solche verschachtelten Strukturen erzeugen kann.

Stell dir vor …

… du packst Geschenke ineinander: Jedes Mal, wenn du außen ein Papier (a) anlegst, musst du später auch wieder ein Papier (b) abwickeln — und zwar in umgekehrter Reihenfolge (zuletzt eingepackt, zuerst ausgepackt). Klammern „(a(a(a)b)b)b", HTML-Tags <div><div>…</div></div> oder Programmier-Blöcke funktionieren genauso. Eine kontextfreie Grammatik ist die Bauanleitung für genau solche „balanced nesting"-Strukturen: Du startest bei einem Symbol S und ersetzt es Schritt für Schritt durch andere Symbole, bis nur noch fertige Wort-Bausteine (Terminale) übrig sind.

Grammatik, Ableitung & Linksableitung im Bild

Definition: Kontextfreie Grammatik
G = (V, Σ, P, S)

V: endliche Menge Nichtterminale (Platzhalter, „Baustellen"). Σ: Eingabealphabet, die Terminale (fertige, nicht mehr ersetzbare Bausteine). P: Produktionsregeln A → β mit A ∈ V und β ∈ (V ∪ Σ)* (β darf beliebig aus Nichtterminalen und Terminalen gemischt sein — auch leer!). S: das Startsymbol, S ∈ V.

VNichtterminale — Platzhalter, werden ersetzt
ΣTerminale — fertige Bausteine, bleiben stehen
PProduktionen A → β — die Ersetzungsregeln
SStartsymbol — hier beginnt jede Ableitung

Der Name „kontextfrei" kommt genau daher, dass links immer nur ein einzelnes Nichtterminal A steht — die Regel gilt unabhängig vom Kontext drumherum (α…γ), im Gegensatz zu allgemeineren Grammatiken, bei denen auch die Umgebung eine Rolle spielen darf.

Direktableitung, Ableitung & die Sprache L(G)
w ⇒ w' falls w = αAγ, w' = αβγ und A → β ∈ P (eine Ersetzung) w ⇒* w' beliebig viele (auch null) Direktableitungen hintereinander L(G) = { w ∈ Σ* | S ⇒* w }

L(G) ist also genau die Menge aller Wörter, die sich — Schritt für Schritt ersetzend — komplett bis auf reine Terminalsymbole aus S herleiten lassen.

Linksableitung — sehr klausurrelevant!

Bei einer Linksableitung ersetzt du in jedem Schritt immer das am weitesten links stehende Nichtterminal — nie eines weiter rechts, egal wie viele Nichtterminale gerade im Wort stehen. Das macht die Ableitung eindeutig nachvollziehbar (es gibt bei fester Regelwahl nur einen Weg, sie aufzuschreiben) und ist in der Klausur fast immer explizit gefordert.

Der Syntaxbaum: Struktur statt flacher Liste

Jede Ableitung entspricht einem Syntaxbaum: Die Wurzel ist das Startsymbol S, innere Knoten sind Nichtterminale, Blätter sind Terminale. Jede Kantengruppe (ein Knoten zu seinen Kindern) entspricht genau einer angewendeten Produktionsregel. Schauen wir uns das an einem kleinen Beispiel an — der Grammatik S → aTe, T → bUd, U → c, die genau das Wort „abcde" erzeugt:

Mehrdeutigkeit: nicht jede Grammatik hat nur EINEN Baum pro Wort

Manche Grammatiken sind mehrdeutig: Für dasselbe Wort gibt es mehrere verschiedene Syntaxbäume (auch wenn beide Linksableitungen zum gleichen Wort führen). Das klassische Beispiel: S → SS | a und das Wort „aaa".

Original-Klausuraufgaben durchgerechnet

🎓 Original-Klausuraufgabe (SoSe 2024, Erstklausur, Aufgabe 3a)

Gib eine kontextfreie Grammatik G₁ für L₅ = { a bi cj di e | i,j ∈ ℕ, i,j ≥ 1 } an. Gib außerdem eine Linksableitung für das Wort abbbccddde an.

Idee zuerst: was muss die Grammatik erzwingen?

Das Wort beginnt fest mit a und endet fest mit e. Dazwischen müssen gleich viele b's wie d's stehen (Exponent i gekoppelt!) und dazwischen beliebig viele c's (Exponent j frei). Ein Nichtterminal T, das bei jedem Rekursionsschritt gleichzeitig ein b links UND ein d rechts anhängt, erzwingt automatisch i(b) = i(d) — genau das, was wir brauchen.

G₁ = (V, Σ, P, S)
V = {S, T, U}
Σ = {a, b, c, d, e}
P = { S → a T e,
      T → b T d | b U d,
      U → U c | c }

S erzeugt zuerst a und e, mit T in der Mitte. T erzeugt rekursiv b/d-Paare (ein b links, ein d rechts pro Rekursionsschritt) und übergibt am Ende an U. U erzeugt rekursiv beliebig viele c's. Struktur: a · bi · cj · di · e.

Linksableitung von „abbbccddde" (i=3, j=2)

  1. Start: S — das Startsymbol ist automatisch das (einzige) linkeste Nichtterminal.
  2. Regel S → aTe: S ⇒ aTe
  3. Regel T → bTd: aTe ⇒ abTde
  4. Regel T → bTd (nochmal): abTde ⇒ abbTdde
  5. Regel T → bUd (Basisfall, T-Rekursion endet): abbTdde ⇒ abbbUddde
  6. Regel U → Uc: abbbUddde ⇒ abbbUcddde
  7. Regel U → c (Basisfall): abbbUcddde ⇒ abbbccddde ✓ — kein Nichtterminal mehr übrig, fertig!
Kontrolle

a·(bbb)·(cc)·(ddd)·e = abbbccddde ✓ — i = 3 (drei b's, drei d's), j = 2 (zwei c's).

🎓 Original-Klausuraufgabe (SoSe 2024, Zweitklausur, Aufgabe 3b)

Gib für die folgende Grammatik G₂ eine mathematische Beschreibung von L(G₂) an:

G₂ = ({S, T, U}, {a, b, c}, P, S)
P = { S → T U T,
      T → a T a | c,
      U → b U b | c }
  1. T analysieren: T → aTa | c erzeugt rekursiv T ⇒* ai c ai — beliebig viele a-Paare symmetrisch um ein zentrales c.
  2. U analysieren: U → bUb | c erzeugt analog U ⇒* bj c bj.
  3. S zusammensetzen: S → T U T verkettet drei unabhängige Teile: erst ein T (Exponent i), dann U (Exponent j), dann noch ein zweites, eigenständiges T (Exponent k — nicht zwangsläufig gleich i!).
Ergebnis
L(G₂) = { a^i c a^i · b^j c b^j · a^k c a^k | i, j, k ∈ ℕ }

ℕ enthält hier auch 0 (T → c bzw. U → c ist der Basisfall mit Exponent 0).

Falle vermieden

Die beiden T-Vorkommen in S → TUT sind zwei getrennte Ableitungs-Instanzen im Syntaxbaum — jede darf ihren eigenen Exponenten wählen. i und k müssen nicht gleich sein, nur weil dieselbe Regel benutzt wird!

Typische Fallen

❌ FalschBei einer Linksableitung darf man Produktionen in beliebiger Reihenfolge auf beliebige Nichtterminale anwenden.
✅ RichtigImmer das am weitesten links stehende Nichtterminal ersetzen — nie eines weiter rechts, auch wenn dort gerade eine „einfachere" Regel anwendbar wäre.
❌ Falschε-Produktionen (A → ε) machen eine Grammatik „nicht mehr kontextfrei".
✅ Richtigε-Produktionen sind bei kontextfreien Grammatiken uneingeschränkt erlaubt (nur in bestimmten Normalformen wie CNF eingeschränkt) — z. B. ist S → aS | ε ganz normal kontextfrei.
❌ FalschEin Syntaxbaum ist nur die Ableitung flach hintereinander aufgeschrieben, z. B. „S, aTe" direkt untereinander.
✅ RichtigDer Syntaxbaum zeigt die Verzweigungsstruktur: S ⇒ aTe bedeutet, S bekommt drei Kinder a, T, e nebeneinander — nicht eine Kette von Einzelknoten.
❌ FalschJede kontextfreie Grammatik hat für jedes Wort genau einen Syntaxbaum.
✅ RichtigManche Grammatiken sind mehrdeutig (S → SS | a ist das Standardbeispiel) — dasselbe Wort kann mehrere verschiedene Syntaxbäume haben.
❌ FalschBei S → TUT sind die Exponenten der beiden T-Vorkommen automatisch gleich, weil es „dieselbe Regel" ist.
✅ RichtigJedes Vorkommen von T im Syntaxbaum ist eine eigenständige Ableitung — nur wenn EIN gemeinsames Nichtterminal (wie T in G₁ oben) für beide Seiten zuständig ist, werden die Exponenten gekoppelt.

Quick-Reference

🎯 Quick-Reference für die Klausur

Grammatik-Bausteine

SymbolBedeutung
VNichtterminale (Platzhalter)
ΣTerminale (fertige Bausteine)
PProduktionen A → β
SStartsymbol
⇒ / ⇒*Direktableitung / beliebig viele Ableitungsschritte
L(G){w ∈ Σ* | S ⇒* w}

Linksableitung — Vorgehen

1. Finde das am weitesten LINKS stehende Nichtterminal.
2. Wende eine passende Regel darauf an.
3. Wiederhole, bis nur noch Terminale übrig sind.
4. Kontrolle: Ergebnis mit dem Zielwort vergleichen.

Merksätze

  • Linksableitung = immer das linkeste Nichtterminal ersetzen, nie beliebig
  • ε-Produktionen sind erlaubt bei CFG (nicht verboten)
  • Syntaxbaum zeigt Verzweigung, nicht die flache Ableitungskette
  • Mehrdeutigkeit ist möglich — ein Wort kann mehrere Syntaxbäume haben
  • Getrennte Vorkommen desselben Nichtterminals (S → TUT) haben unabhängige Exponenten, außer eine gemeinsame Regel koppelt sie explizit

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.Was bedeutet eine Produktion A → β in einer KONTEXTFREIEN Grammatik?

Genau das macht eine Grammatik „kontextfrei": links steht immer nur ein einzelnes Nichtterminal A, das ersetzt werden darf — egal in welcher Umgebung es steht.

2.Wie schreibt man kurz die Menge aller vom Startsymbol S ableitbaren Wörter (die „Sprache von G")?

L(G) = {w ∈ Σ* | S ⇒* w}.

3.Aktuelle Ableitungszeile: „aTbUc" (T und U sind Nichtterminale). Welches Nichtterminal wird im NÄCHSTEN Schritt einer Linksableitung ersetzt?

Linksableitung heißt: immer das am weitesten links stehende Nichtterminal ersetzen. Das ist hier T — U kommt erst danach dran.

4.🎓 Klausur EK 3a: Für L₅ = {abicjdie} und das Wort abbbccddde — wie groß ist der Exponent i (Anzahl b's = Anzahl d's)?

Drei b's und drei d's im Wort abbbccddde → i = 3.

5.🎓 Klausur EK 3a: Und wie groß ist der Exponent j (Anzahl c's) im selben Wort abbbccddde?

Zwei c's im Wort → j = 2.

6.🎓 Klausur ZK 3b: Für G₂ mit S → TUT, T → aTa|c, U → bUb|c — was erzeugt T (für sich allein)?

T → aTa | c erzeugt rekursiv a^i c a^i — bei jedem Rekursionsschritt kommt links UND rechts ein a dazu, bis die Basis T → c greift.
