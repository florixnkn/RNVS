Worum geht's?

Ein Hypothesentest ist ein formales Verfahren, um zu entscheiden, ob deine Daten gegen eine Behauptung sprechen — oder nicht. Du fällst nie eine 100%-sichere Entscheidung, sondern kontrollierst nur, wie oft du dich irrst, wenn du diese Prozedur immer wieder anwendest.

Stell dir vor …

… ein Gerichtsverfahren. Die Nullhypothese H₀ ist die Unschuldsvermutung: "Der Angeklagte ist unschuldig", bis das Gegenteil bewiesen ist. Die Alternativhypothese H₁ ist "schuldig". Das Gericht sammelt Beweise (Daten) und entscheidet dann: reicht die Beweislage, um H₀ zu verwerfen? Zwei Dinge können dabei schiefgehen: ein Unschuldiger wird verurteilt (Fehler 1. Art) oder ein Schuldiger geht frei (Fehler 2. Art). Das Signifikanzniveau α ist die Grenze, wie hoch du das Risiko für den ersten, schlimmeren Fehler zulässt.

Hypothesentests im Bild

Aufbau eines Tests
H₀: Nullhypothese (zu falsifizierende Behauptung) H₁: Alternativhypothese Teststatistik: T = T(X₁,...,Xₙ) Ablehnungsbereich: K ⊆ ℝ Entscheidung: Lehne H₀ ab, wenn T ∈ K

Fehlerarten: das Gerichtsverfahren als Matrix

Der z-Test (Gaußtest): μ bei bekanntem σ

Formel: Teststatistik & Ablehnungsbereiche
Z = (X̄ₙ − μ₀) / (σ/√n) ~ N(0,1) unter H₀ Zweiseitig (H₁: μ ≠ μ₀): K = { |Z| > z_{α/2} } Einseitig (H₁: μ > μ₀): K = { Z > z_α } Einseitig (H₁: μ < μ₀): K = { Z < −z_α }

Der p-Wert

Formel & Entscheidungsregel
p-Wert = P(T mindestens so extrem wie T_beob | H₀ wahr) Entscheidungsregel: H₀ ablehnen ⟺ p-Wert ≤ α

Der p-Wert ist keine Wahrscheinlichkeit dafür, dass H₀ wahr ist — er beantwortet eine andere Frage (siehe "Typische Fallen").

Chi-Quadrat-Test (kurz)

Prüft, ob beobachtete Häufigkeiten zu einem angenommenen Modell passen (Anpassungstest) oder ob zwei kategoriale Merkmale unabhängig sind (Unabhängigkeitstest):

Formel: Chi-Quadrat-Anpassungstest
eᵢ = n · p₀ᵢ (erwartete Häufigkeit unter H₀) χ² = Σᵢ (nᵢ − eᵢ)² / eᵢ ~ χ²_{K−1} unter H₀ Ablehnungsbereich: χ² > χ²_{K−1, α}

Faustregel: alle eᵢ ≥ 5, sonst wird die Approximation unzuverlässig.

Der Unabhängigkeitstest funktioniert nach demselben Prinzip, prüft aber, ob zwei kategoriale Merkmale in einer Kontingenztabelle unabhängig voneinander sind:

Formel: Chi-Quadrat-Unabhängigkeitstest
eᵢⱼ = nᵢ. · n.ⱼ / n (erwartet unter Unabhängigkeit) χ² = Σᵢ Σⱼ (nᵢⱼ − eᵢⱼ)² / eᵢⱼ ~ χ²_{(R−1)(C−1)}

R = Anzahl Zeilen, C = Anzahl Spalten der Kontingenztabelle.

Schritt-für-Schritt: Beispiele durchgerechnet

Durchgerechnetes Beispiel: z-Test zweiseitig

Ein Hersteller behauptet, seine Glühbirnen halten im Schnitt μ₀ = 500 Stunden (σ = 20 Stunden bekannt). Eine Stichprobe von n = 25 Birnen ergibt X̄ = 490 Stunden. Test bei α = 0.05: H₀: μ = 500 vs. H₁: μ ≠ 500.

  1. Teststatistik berechnen: Z = (X̄ − μ₀)/(σ/√n) = (490 − 500)/(20/√25) = −10/4 = −2.5
  2. Kritischen Wert bestimmen: zweiseitig, α = 0.05 → z_{0.025} = 1.96
  3. Vergleichen: |Z| = 2.5 > 1.96 → Z liegt im Ablehnungsbereich
  4. Entscheidung: H₀ wird abgelehnt — die mittlere Lebensdauer weicht signifikant von 500 Stunden ab
  5. p-Wert (zur Kontrolle): p = 2·(1−Φ(2.5)) = 2·(1−0.99379) = 2·0.00621 ≈ 0.0124 — kleiner als α = 0.05, bestätigt die Ablehnung

Bei n = 200 Nutzern wird erfasst, welche App-Version sie haben (A = alt, n=180; Ā = neu, n=20) und ob ihr Feedback positiv oder negativ war. Sind Version und Feedback unabhängig?

Typische Fallen

❌ Falschp-Wert = 0.03 bedeutet: H₀ ist mit 3% Wahrscheinlichkeit wahr.
✅ RichtigDer p-Wert ist P(Daten so extrem oder extremer | H₀ wahr) — eine Aussage über die Daten, nicht über die Wahrscheinlichkeit von H₀ selbst. Das ist das häufigste Missverständnis überhaupt.
❌ FalschImmer z_{α/2} als kritischen Wert verwenden.
✅ RichtigZweiseitig (H₁: ≠) → z_{α/2} (z. B. 1.96 bei α=0.05). Einseitig (H₁: > oder <) → z_α (z. B. 1.645 bei α=0.05) — nicht verwechseln!
❌ FalschFehler 1. und 2. Art durcheinanderbringen.
✅ RichtigFehler 1. Art (α): H₀ ablehnen, obwohl sie wahr ist. Fehler 2. Art (β): H₀ beibehalten, obwohl H₁ wahr ist. Trennschärfe = 1−β.
❌ FalschBeim Anpassungstest mit K Kategorien: df = K.
✅ Richtigdf = K − 1 (minus 1 wegen der Normierung Σnᵢ=n). Werden zusätzlich Parameter aus den Daten geschätzt, zieht man für jeden weitere 1 ab.
❌ Falschσ ist unbekannt, aber ich rechne trotzdem mit dem Gaußtest (z-Test).
✅ Richtigσ unbekannt → t-Test mit T = (X̄−μ₀)/(S/√n), verteilt nach t_{n−1}, nicht nach N(0,1).

Merksätze

  • H₀ ablehnen ⟺ p-Wert ≤ α — nicht "H₀ ist unwahrscheinlich"
  • α = P(Fehler 1. Art), β = P(Fehler 2. Art), Power = 1 − β
  • Zweiseitig → z_{α/2}, einseitig → z_α
  • χ²-Test: df = Anzahl Kategorien − 1 (minus geschätzte Parameter)
  • σ unbekannt ⟹ t-Verteilung statt Normalverteilung

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.🧮 Klausurtyp: H₀: μ=500, n=25, X̄=490, σ=20. Wie lautet die Teststatistik Z?

Z = (490−500)/(20/√25) = −10/4 = −2.5.

2.Gleiches Beispiel, α = 0.05, zweiseitig: Wird H₀ abgelehnt?

|Z| = 2.5 liegt über dem kritischen Wert z_{0.025} = 1.96 → Z fällt in den Ablehnungsbereich, H₀ wird verworfen.

3.Wie groß ist der p-Wert für Z = −2.5 (zweiseitig)?

p = 2·(1−Φ(2.5)) = 2·0.00621 ≈ 0.0124.

4.Was sagt ein p-Wert von 0.03 korrekt aus?

Der p-Wert ist eine bedingte Wahrscheinlichkeit über die Daten gegeben H₀ — keine Aussage über die Wahrscheinlichkeit von H₀ oder H₁.

5.Wie heißt der Fehler, H₀ abzulehnen, obwohl sie in Wirklichkeit wahr ist?

Das ist der Fehler 1. Art — Wahrscheinlichkeit α, das Signifikanzniveau.

6.Würfel-Beispiel (n=120, 6 Kategorien, eᵢ=20): Wie groß ist χ²?

Summe der (nᵢ−eᵢ)²/eᵢ über alle 6 Kategorien = 38/20 = 1.9.

7.Anpassungstest mit K = 6 Kategorien: wie viele Freiheitsgrade hat χ²?

df = K − 1 = 6 − 1 = 5 (ein Freiheitsgrad geht durch die Normierung Σnᵢ=n verloren).
