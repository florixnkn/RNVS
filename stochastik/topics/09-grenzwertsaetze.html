Worum geht's?

Was passiert, wenn du ein Zufallsexperiment nicht einmal, sondern sehr oft wiederholst? Drei Sätze beantworten das aus verschiedenen Blickwinkeln: Die Tschebyschew-Ungleichung gibt eine (grobe) Garantie für jede beliebige Verteilung, das Gesetz der großen Zahlen sagt, dass sich dein Stichprobenmittel auf den wahren Erwartungswert einpendelt, und der zentrale Grenzwertsatz verrät sogar, welche Form die Verteilung der Summe annimmt: eine Glockenkurve.

Stell dir vor …

… du wirfst eine faire Münze. Nach 10 Würfen kann das Verhältnis Kopf:Zahl locker 7:3 sein — reiner Zufall. Nach 10.000 Würfen wird es fast immer sehr nah an 50:50 liegen. Das ist das Gesetz der großen Zahlen in Aktion: Je mehr Würfe, desto näher rückt die relative Häufigkeit an die wahre Wahrscheinlichkeit 0.5. Zählst du stattdessen die absolute Anzahl der Köpfe, wächst die mit jedem Wurf weiter — die Summe konvergiert nicht, nur der Durchschnitt tut es.

Tschebyschew, GGZ & ZGWS im Bild

Ausgangspunkt für alle drei Sätze: X₁, X₂, … sind unabhängig und identisch verteilt (i.i.d.) mit E[Xᵢ] = μ und Var(Xᵢ) = σ² (endlich). Definiert werden das Stichprobenmittel X̄ₙ = (X₁+…+Xₙ)/n und die Partialsumme Sₙ = X₁+…+Xₙ.

Tschebyschew-Ungleichung: eine Garantie für jede Verteilung

Egal wie X verteilt ist — solange E[X] und Var(X) existieren, begrenzt die Tschebyschew-Ungleichung, wie wahrscheinlich eine große Abweichung vom Mittelwert ist:

Formel: Tschebyschew-Ungleichung
P(|X − μ| ≥ ε) ≤ σ²/ε² für alle ε > 0 Äquivalent: P(|X − μ| < ε) ≥ 1 − σ²/ε²

Braucht nur E[X] und Var(X) — funktioniert für jede Verteilung mit endlicher Varianz, liefert aber oft eine sehr grobe (pessimistische) Schranke.

Gesetz der großen Zahlen: das Mittel stabilisiert sich

Wiederholst du ein Experiment sehr oft und bildest den laufenden Durchschnitt X̄ₙ, schwankt dieser anfangs stark und wird mit wachsendem n immer ruhiger — er konvergiert gegen μ.

Formel: Gesetz der großen Zahlen (GGZ)
Schwaches GGZ: X̄ₙ →^P μ d.h. P(|X̄ₙ − μ| ≥ ε) → 0 für n→∞ Starkes GGZ: X̄ₙ →^{f.s.} μ d.h. P(lim_{n→∞} X̄ₙ = μ) = 1

Beweisidee des schwachen GGZ über Tschebyschew: P(|X̄ₙ−μ|≥ε) ≤ Var(X̄ₙ)/ε² = σ²/(n·ε²) → 0.

Zentraler Grenzwertsatz: die Summe wird zur Glockenkurve

Der ZGWS geht noch weiter als das GGZ: Er sagt nicht nur, dass sich X̄ₙ stabilisiert, sondern auch, welche Form die (standardisierten) Abweichungen annehmen — nämlich näherungsweise eine Standardnormalverteilung, unabhängig davon, wie X selbst verteilt ist.

Formel: Zentraler Grenzwertsatz (ZGWS)
(X̄ₙ − μ) / (σ/√n) →^d N(0,1) für n → ∞ Approximation: X̄ₙ ≈ N(μ, σ²/n) Sₙ ≈ N(n·μ, n·σ²)

Faustregel: n ≥ 30 reicht meist aus; bei stark schiefen Ausgangsverteilungen braucht es mehr.

🎓 Nachklausur SS 2025 — ZGWS auf eine Summe angewendet

X₁,…,X₈₁ i.i.d. mit E[Xᵢ] = μ = 20, Var(Xᵢ) = σ² = 49. Gesucht: P(S₈₁ ≥ 1743.5).

E(S₈₁) = 81·20 = 1620,   sd(S₈₁) = √(81·49) = √3969 = 63.

P(S₈₁ ≥ 1743.5) = P(Z ≥ (1743.5−1620)/63) = P(Z ≥ 1.96) = 1 − Φ(1.96) = 0.025.

Normalapproximation der Binomialverteilung

Eine der wichtigsten Anwendungen des ZGWS: X ~ B(n,p) lässt sich für großes n (Faustregel: np ≥ 5 und n(1−p) ≥ 5) durch eine Normalverteilung annähern.

Formel: Normalapproximation
X ~ B(n,p), μ = np, σ² = np(1−p) X ≈ N(np, np(1−p)) Standardisierung: Z = (X − np) / √(np(1−p)) ≈ N(0,1) Mit Stetigkeitskorrektur (genauer): P(X ≤ k) ≈ Φ((k + 0.5 − np) / √(np(1−p)))
f.s. ⟹ WK ⟹ Vert.Hierarchie der Konvergenzbegriffe
n ≥ 30Faustregel für ZGWS-Approximation
np≥5, n(1−p)≥5Faustregel für Normalapprox. von B(n,p)
σ² muss endlich seinsonst gilt der ZGWS nicht (z.B. Cauchy)

Schritt-für-Schritt: Beispiele

Beispiel 1: Tschebyschew-Schranke berechnen

Gegeben: E[X] = μ = 50, Var(X) = σ² = 25. Gesucht: eine Schranke für P(|X − 50| ≥ 10).

  1. ε identifizieren: ε = 10 (die Abweichung, die uns interessiert).
  2. Formel einsetzen:
    P(|X − 50| ≥ 10) ≤ σ²/ε² = 25/10² = 25/100 = 0.25
Zur Einordnung

Die Tschebyschew-Schranke sagt: Die Wahrscheinlichkeit einer Abweichung von mindestens 10 ist höchstens 25% — sie könnte in Wirklichkeit deutlich kleiner sein. Das ist der Preis dafür, dass die Schranke für jede Verteilung mit dieser Varianz gilt.

Beispiel 2: Mindeststichprobengröße aus Tschebyschew

Gegeben: σ² = 1. Wie groß muss n mindestens sein, damit P(|X̄ₙ − μ| ≥ 0.1) ≤ 0.05 gilt?

  1. Var(X̄ₙ) einsetzen: Var(X̄ₙ) = σ²/n = 1/n, also
    P(|X̄ₙ − μ| ≥ 0.1) ≤ σ²/(n·0.1²) = 1/(0.01·n)
  2. Bedingung aufstellen: 1/(0.01·n) ≤ 0.05
  3. Nach n auflösen:
    n ≥ 1/(0.01·0.05) = 1/0.0005 = 2000

🎓 Nachklausur SS 2025: Stichprobenumfang bestimmen — zwei Wege

Gegeben: σ² = 49. Gesucht: kleinstes n, sodass P(|X̄ₙ − μ| ≤ 0.5) > 0.9 gilt. Diesmal auf zwei Arten gelöst — einmal konservativ über Tschebyschew, einmal schärfer über den ZGWS.

  1. Weg 1 — Tschebyschew (gilt immer, ist aber grob):
    P(|X̄ₙ−μ| ≥ 0.5) ≤ σ²/(n·0.5²) = 49/(0.25n). Bedingung: 1 − (49/n)/0.25 > 0.9 ⟹ (49/n)/0.25 < 0.1 ⟹ n > 49/(0.1·0.25) = 1960 ⟹ n ≥ 1961
  2. Weg 2 — ZGWS (schärfere Näherung):
    Mit X̄ₙ ≈ N(μ, σ²/n) gilt P(|X̄ₙ−μ| ≤ ε) ≈ 2Φ(ε√n/σ) − 1 > 0.9 ⟹ ε√n/σ > z₀.₉₅ = 1.645 ⟹ n > (z₀.₉₅·σ/ε)² = (1.645·7/0.5)² ≈ 530.4 ⟹ n = 531
Merksatz

Tschebyschew gilt immer (für jede Verteilung mit endlicher Varianz), ist aber grob — hier n ≥ 1961. Die ZGWS-Näherung ist schärfer und liefert mit n = 531 einen deutlich kleineren, realistischeren Stichprobenumfang.

🧮 Klausurtyp: ZGWS auf eine Summe anwenden

X₁,…,X₁₀₀ sind i.i.d. mit E[Xᵢ] = 2, Var(Xᵢ) = 9. Gesucht: P(S₁₀₀ > 210).

  1. ZGWS-Approximation für die Summe:
    S₁₀₀ ≈ N(100·2, 100·9) = N(200, 900), also σ_S = √900 = 30
  2. Standardisieren:
    P(S₁₀₀ > 210) = P(Z > (210−200)/30) = P(Z > 1/3) = P(Z > 0.33)
  3. Φ-Tabelle nutzen:
    P(Z > 0.33) = 1 − Φ(0.33) ≈ 1 − 0.629 = 0.371

🧮 Klausurtyp: Normalapproximation einer Binomialverteilung

X ~ B(100, 0.4). Gesucht: P(X ≤ 35).

  1. μ und σ bestimmen:
    μ = 100·0.4 = 40,   σ = √(100·0.4·0.6) = √24 ≈ 4.899
  2. Standardisieren und Φ ablesen (ohne Stetigkeitskorrektur):
    P(X ≤ 35) ≈ Φ((35−40)/4.899) = Φ(−1.02) = 1 − Φ(1.02) ≈ 0.154
  3. Mit Stetigkeitskorrektur (genauer):
    P(X ≤ 35) ≈ Φ((35.5−40)/4.899) = Φ(−0.918) ≈ 0.179

Typische Fallen

❌ FalschDas GGZ sagt, dass die Summe Sₙ gegen μ konvergiert.
✅ RichtigEs ist das Mittel X̄ₙ = Sₙ/n, das gegen μ konvergiert. Die Summe Sₙ selbst wächst mit n → ∞ unbegrenzt weiter.
❌ FalschDer ZGWS gilt für alle Verteilungen.
✅ RichtigDer ZGWS braucht endliche Varianz σ². Bei der Cauchy-Verteilung z.B. existiert σ² nicht — der ZGWS gilt dort nicht.
❌ FalschZ = (X̄ₙ − μ) / (σ²/√n)
✅ RichtigZ = (X̄ₙ − μ) / (σ/√n) — im Nenner steht die Standardabweichung σ, nicht die Varianz σ²!
❌ FalschP(|X − μ| ≥ ε) = σ²/ε² (exakter Wert)
✅ RichtigTschebyschew liefert nur eine obere Schranke: P(|X−μ|≥ε) ≤ σ²/ε². Der tatsächliche Wert kann deutlich kleiner sein.

Quick-Reference

🎯 Quick-Reference für die Klausur

Kernformeln

P(|X−μ| ≥ ε) ≤ σ²/ε²           Tschebyschew (jede Verteilung mit endl. Varianz)
X̄ₙ →^P μ                       Schwaches GGZ
X̄ₙ →^{f.s.} μ                   Starkes GGZ
(X̄ₙ−μ)/(σ/√n) →^d N(0,1)       Zentraler Grenzwertsatz
X̄ₙ ≈ N(μ, σ²/n)                ZGWS-Approximation fürs Mittel
Sₙ ≈ N(nμ, nσ²)                 ZGWS-Approximation für die Summe
B(n,p) ≈ N(np, np(1−p))         Normalapproximation der Binomialvert.

Konvergenzbegriffe im Vergleich

BegriffSymbolBedeutung
Konvergenz in Wahrscheinlichkeit→^PP(|X̄ₙ−μ| ≥ ε) → 0
Fast sichere Konvergenz→^{f.s.}P(lim X̄ₙ = μ) = 1
Konvergenz in Verteilung→^dF_{Xₙ}(x) → F(x)

Hierarchie: fast sicher ⟹ in Wahrscheinlichkeit ⟹ in Verteilung

Merksätze

  • Mittel konvergiert, Summe wächst — GGZ gilt für X̄ₙ, nicht für Sₙ
  • Tschebyschew ist eine Schranke, kein exakter Wert
  • ZGWS braucht endliches σ² und i.i.d.
  • Standardisierung immer mit σ (nicht σ²) im Nenner
  • Normalapproximation von B(n,p) nur bei genügend großem n (np≥5, n(1−p)≥5)

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.μ = 50, σ² = 25. Wie groß ist die Tschebyschew-Schranke für P(|X−50| ≥ 10)?

σ²/ε² = 25/10² = 25/100 = 0.25.

2.Var(X) = 8. Wie groß ist die Tschebyschew-Schranke für P(|X−μ| ≥ 4)?

σ²/ε² = 8/4² = 8/16 = 0.5.

3.Was besagt das Gesetz der großen Zahlen?

Das GGZ betrifft das arithmetische Mittel X̄ₙ, das gegen μ konvergiert — nicht die Summe (die wächst) und nicht die Verteilungsform (das ist der ZGWS).

4.σ² = 1. Wie groß muss n mindestens sein, damit P(|X̄ₙ−μ| ≥ 0.1) ≤ 0.05 gilt (Tschebyschew)?

n ≥ σ²/(ε²·δ) = 1/(0.1²·0.05) = 1/0.0005 = 2000.

5.Wie heißt der Satz, der besagt, dass Summen vieler unabhängiger Zufallsvariablen näherungsweise normalverteilt sind? (Abkürzung reicht)

Der zentrale Grenzwertsatz (ZGWS).

6.Was liefert die Tschebyschew-Ungleichung?

Tschebyschew liefert eine obere Schranke — gültig für jede Verteilung mit endlicher Varianz, aber oft nicht besonders scharf.

7.🧮 Klausurtyp: X ~ B(100, 0.4). Wie groß ist σ = √(np(1−p)) für die Normalapproximation?

σ = √(100·0.4·0.6) = √24 ≈ 4.9.

8.🎓 Nachklausur SS25: S₈₁ mit E=1620, sd=63. Was ist P(S₈₁ ≥ 1743.5)? (Tipp: standardisieren, Φ(1.96)=0.975)

(1743.5−1620)/63 = 1.96. P(Z≥1.96) = 1 − 0.975 = 0.025.

9.🎓 Nachklausur SS25: Wie groß muss n mindestens sein (ZGWS-Weg, z₀.₉₅=1.645, σ=7, ε=0.5)?

n > (1.645·7/0.5)² = 23.03² ≈ 530.4 ⟹ n = 531 (aufgerundet).
