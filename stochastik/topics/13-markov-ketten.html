Worum geht's?

Eine Markov-Kette ist ein Zufallsprozess mit einer sehr angenehmen Eigenschaft: Um vorherzusagen, was als Nächstes passiert, musst du nur den aktuellen Zustand kennen — die komplette Vorgeschichte ist irrelevant. Das nennt man Gedächtnislosigkeit.

Stell dir vor …

… das Wetter von morgen. Ob es morgen regnet, hängt (in diesem vereinfachten Modell) nur davon ab, ob es heute regnet oder sonnig ist — nicht davon, wie das Wetter letzte Woche war. "Heute sonnig" plus eine feste Übergangsregel reicht, um die Wahrscheinlichkeit für "morgen sonnig" zu berechnen. Diese Übergangsregel — mit welcher Wahrscheinlichkeit du von einem Zustand in einen anderen wechselst — ist das Herzstück jeder Markov-Kette: die Übergangsmatrix.

Übergangsmatrix & stationäre Verteilung im Bild

Definition: Markov-Eigenschaft
P(X_{n+1}=j | X₀=i₀,...,X_n=iₙ) = P(X_{n+1}=j | X_n=iₙ)

"Nur der aktuelle Zustand bestimmt die Zukunft — die Vergangenheit ist bedingt irrelevant."

Die Übergangsmatrix P sammelt alle Übergangswahrscheinlichkeiten p_ij = P(X_{n+1}=j | X_n=i). Zwei Eigenschaften muss jede gültige Übergangsmatrix erfüllen: alle Einträge ≥ 0, und jede Zeile summiert sich zu 1 (von jedem Zustand aus musst du ja irgendwohin wechseln — auch "bleiben" zählt als Übergang).

Unser Wetter-Beispiel für den Rest der Seite: zwei Zustände, Sonnig (S) und Regen (R), mit Übergangsmatrix:

Übergangsmatrix P
S R S ( 0.9 0.1 ) R ( 0.5 0.5 )

Von Sonnig aus bleibt es zu 90% sonnig, springt zu 10% auf Regen. Von Regen aus ist es 50/50. Zeilensummen: 0.9+0.1=1 ✓, 0.5+0.5=1 ✓.

Mehrschrittige Übergänge = Matrixpotenz

Chapman-Kolmogorov
p_ij^(n) = P(X_{n+m}=j | X_m=i) = [P^n]_ij μ^(n) = μ^(0) · P^n (Verteilung nach n Schritten, μ^(0) = Startverteilung)

P^n ist die n-te Matrizenpotenz von P (Matrixmultiplikation!) — nicht n·P.

Stationäre Verteilung: der Gleichgewichtszustand

Definition & Grenzwertsatz
π ist stationär, wenn: π · P = π, Σⱼ πⱼ = 1, πⱼ ≥ 0 Für irreduzible, aperiodische Ketten: μ^(n) → π für n → ∞ (unabhängig vom Start!)

π ist ein Linkseigenvektor von P zum Eigenwert 1 — π steht links, nicht rechts von P.

Irreduzibelalle Zustände erreichen sich gegenseitig
Aperiodischkein fester Rhythmus, d(i)=1
Absorbierendp_ii=1 — Zustand wird nie verlassen
RekurrentRückkehr zum Zustand ist sicher (WK 1)

Schritt-für-Schritt: Beispiele durchgerechnet

Durchgerechnetes Beispiel: 2-Schritt-Wahrscheinlichkeit

Wie hoch ist die Wahrscheinlichkeit, dass es übermorgen regnet, wenn es heute sonnig ist? Gesucht: [P²]_{S,R}.

  1. P² = P · P aufstellen (Matrizenprodukt, nicht Skalarmultiplikation!)
  2. [P²]_{S,S} berechnen: 0.9·0.9 + 0.1·0.5 = 0.81 + 0.05 = 0.86
  3. [P²]_{S,R} berechnen: 0.9·0.1 + 0.1·0.5 = 0.09 + 0.05 = 0.14
  4. Zur Kontrolle — [P²]_{R,S} und [P²]_{R,R}: 0.5·0.9+0.5·0.5 = 0.70, 0.5·0.1+0.5·0.5 = 0.30. Zeilensummen: 0.86+0.14=1 ✓, 0.70+0.30=1 ✓
Ergebnis

P(übermorgen Regen | heute sonnig) = [P²]_{S,R} = 0.14 = 14%.

Durchgerechnetes Beispiel: Stationäre Verteilung

Wie sieht das Wetter langfristig aus, unabhängig davon, wie es heute war?

  1. Gleichungssystem π·P = π aufstellen:
    π_S = 0.9·π_S + 0.5·π_R  ⟹  0.1·π_S = 0.5·π_R  ⟹  π_S = 5·π_R
    (die zweite Gleichung π_R = 0.1·π_S + 0.5·π_R ist linear abhängig von der ersten — wie immer bei 2 Zuständen)
  2. Normierung statt der abhängigen Gleichung: π_S + π_R = 1
  3. Einsetzen & auflösen: 5·π_R + π_R = 1 ⟹ 6·π_R = 1 ⟹ π_R = 1/6 ≈ 0.1667, π_S = 5·(1/6) = 5/6 ≈ 0.8333
  4. Probe: π_S·0.9 + π_R·0.5 = (5/6)·0.9 + (1/6)·0.5 = 0.75 + 0.0833 = 0.8333 = π_S ✓
Interpretation

Egal ob es heute sonnig oder regnerisch ist — langfristig ist es an 5/6 ≈ 83.3% der Tage sonnig und an 1/6 ≈ 16.7% der Tage regnerisch. Diese Konvergenz gilt, weil die Kette irreduzibel ist (S und R erreichen sich gegenseitig) und aperiodisch (p_SS = 0.9 > 0 reicht dafür bereits aus).

Schritt nP(sonnig) bei Start "heute sonnig"
n = 0 (heute)1.0
n = 10.9
n = 20.86
n → ∞0.833 (= π_S)

Typische Fallen

❌ FalschP² = 2·P
✅ RichtigP² = P·P — Matrizenprodukt, keine Skalarmultiplikation!
❌ FalschP·π = π (π von rechts multiplizieren)
✅ Richtigπ·P = π — π ist ein Zeilenvektor links von P (Linkseigenvektor zum Eigenwert 1).
❌ FalschÜbergangsmatrix nicht auf Zeilensummen geprüft.
✅ RichtigImmer prüfen: Σⱼ p_ij = 1 für jede Zeile i. Stimmt das nicht, ist die Matrix fehlerhaft.
❌ FalschJede irreduzible Markov-Kette konvergiert gegen ihre stationäre Verteilung.
✅ RichtigNur irreduzible UND aperiodische Ketten konvergieren. Periodische Ketten oszillieren stattdessen für immer.
❌ Falschμ^(2) = μ^(0)·P² gilt immer, auch wenn kein μ^(0) gegeben ist.
✅ Richtigp_ij^(n) = [P^n]_ij ist die bedingte WK bei Start in i. μ^(n) = μ^(0)·P^n brauchst du nur, wenn eine konkrete unbedingte Startverteilung gegeben ist.

Zustände klassifizieren: rekurrent, transient, absorbierend

Fast jede Markov-Klausuraufgabe fragt irgendwann: Ist dieser Zustand rekurrent, transient oder absorbierend? Ist die Kette irreduzibel? Diese Begriffe beschreiben, was langfristig mit einem Zustand passiert — unabhängig von den konkreten Zahlenwerten.

Rekurrentman kommt immer wieder zurück — Rückkehr ist sicher (WK 1)
Transientirgendwann für immer weg — Rückkehr ist nicht sicher
Absorbierend"Hotel California": einchecken, nie wieder auschecken (p_ii=1)
Irreduzibeljeder Zustand ist von jedem Zustand aus erreichbar
Merksatz (Klausur-Gold)

In einer endlichen irreduziblen Kette sind alle Zustände (positiv) rekurrent. Ist die Kette dagegen nicht irreduzibel, lohnt sich der Blick auf die Zerlegung des Zustandsraums.

Zerlegung des Zustandsraums
Zustandsraum = T ∪ C₁ ∪ C₂ ∪ ... T: transiente Zustände (Menge, aus der man irgendwann für immer herausfällt) Cₖ: irreduzible, geschlossene Klassen (einmal drin, nie mehr raus — innen ist alles rekurrent)

"Geschlossen" heißt: aus Cₖ führt kein Pfeil mehr heraus. Ein absorbierender Zustand ist der Spezialfall einer Klasse Cₖ mit nur einem Element.

🎓 Klausur SS 2024: Gen-Modell (Zustandsklassifikation)

Ein Gen kann drei Zustände annehmen: Aktiv (A), Inaktiv (I) oder Mutiert (M). Ist das Gen einmal mutiert, bleibt es das für immer: p_MM = 1.

Übergangsmatrix P
A I M A ( 0.6 0.3 0.1 ) I ( 0.4 0.5 0.1 ) M ( 0 0 1 )
  1. M ist absorbierend: p_MM = 1 — von M führt kein Pfeil mehr weg.
  2. Irreduzibilität prüfen: Aus M kommt man nie wieder heraus, M erreicht also weder A noch I zurück. Die Kette ist damit nicht irreduzibel.
  3. Langfristiges Verhalten: A und I wechseln mit positiver Wahrscheinlichkeit nach M (p_AM=0.1, p_IM=0.1), und M wird nie wieder verlassen — die Kette landet für t→∞ fast sicher in M.
  4. Zerlegung: T = {A, I} (transient — beide werden irgendwann für immer verlassen), C₁ = {M} (absorbierende Klasse mit einem Element).

Zur Kontrolle die zweite Matrixpotenz (aus der Klausur gegeben):

P² (gerundet)
A I M A ( 0.48 0.33 0.19 ) I ( 0.44 0.37 0.19 ) M ( 0 0 1 )

Startet die Kette sicher in A (μ^(0) = (1,0,0)), dann ist μ^(2) = μ^(0)·P² gerade die erste Zeile von P²: (0.48, 0.33, 0.19). Bei t=2 ist Aktiv mit 0.48 am wahrscheinlichsten — trotzdem "leckt" schon 19% Wahrscheinlichkeitsmasse unwiderruflich nach M ab.

🎓 Nachklausur SS 2025: Kundenpräferenz (kompakt)

Drei Marken A, U, B. Direkte Wechsel A↔B sind laut Angabe nicht möglich, aber beide kommunizieren über U (A→U→B und B→U→A sind möglich). Damit erreicht jeder Zustand jeden anderen — die Kette ist irreduzibel, also sind (endlich + irreduzibel) alle drei Zustände rekurrent. Zerlegung: T = ∅, C₁ = {A, U, B} — eine einzige rekurrente Klasse, die den gesamten Zustandsraum umfasst.

❌ FalschBeim Vervollständigen einer Übergangsmatrix eine Spalte auf 1 auffüllen.
✅ RichtigEs ist immer die Zeile, die sich zu 1 summiert. Im NK25-Beispiel folgt aus "direkter Übergang A↔B nicht möglich" sofort p_AB = p_BA = 0 — die restlichen Einträge holt man sich dann über Zeilensumme = 1.

🎓 Klausur SS 2025 (Bonus): Pfadwahrscheinlichkeit über einen Zwischenschritt

Formel
P(X₀=i, X₂=i) = P(X₀=i) · Σₛ p_is · p_si

Erst die Startwahrscheinlichkeit, dann über alle möglichen Zwischenzustände s summieren.

Beispiel: Zustände U, V, W mit P = [[0.7,0.3,0],[0.3,0.5,0.2],[0.4,0.5,0.1]], μ^(0) = (0.5, 0.3, 0.2). Gesucht: P(X₀=V, X₂=V).

  1. Startwahrscheinlichkeit: P(X₀=V) = μ^(0)(V) = 0.3
  2. Summe über Zwischenzustände: Σₛ p_Vs·p_sV = p_VU·p_UV + p_VV·p_VV + p_VW·p_WV = 0.3·0.3 + 0.5·0.5 + 0.2·0.5 = 0.09+0.25+0.10 = 0.44
  3. Multiplizieren: P(X₀=V, X₂=V) = 0.3 · 0.44 = 0.132

Merksätze

  • P^n ist Matrixpotenz — Matrizenprodukt, nicht n·P

  • Bei 2 Zuständen ist eine der beiden π·P=π-Gleichungen immer redundant → durch Σπⱼ=1 ersetzen

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.P = [[0.9,0.1],[0.5,0.5]] (S,R). Wie hoch ist [P²]_{S,R} — die Wahrscheinlichkeit, in 2 Schritten von Sonnig nach Regen zu wechseln?

0.9·0.1 + 0.1·0.5 = 0.09 + 0.05 = 0.14.

2.🧮 Klausurtyp: Wie groß ist π_S (stationärer Anteil "sonnig") für dieselbe Matrix?

π_S = 5/6 ≈ 0.833 (aus 0.1π_S = 0.5π_R und π_S+π_R=1).

3.Und wie groß ist π_R?

π_R = 1/6 ≈ 0.167 (π_S + π_R = 1, π_S = 5·π_R).

4.Wie ist die stationäre Verteilung π richtig definiert?

π ist ein Linkseigenvektor von P zum Eigenwert 1: π·P = π.

5.Was müssen die Zeilensummen einer gültigen Übergangsmatrix sein?

Von jedem Zustand aus muss man irgendwohin wechseln (auch "bleiben" zählt) — daher Σⱼ p_ij = 1 für jede Zeile.

6.Wie heißt die Eigenschaft, dass nur der aktuelle Zustand die Zukunft bestimmt — die Vergangenheit aber nicht?

Die Markov-Eigenschaft, umgangssprachlich auch "Gedächtnislosigkeit" genannt.

7.Unter welcher Bedingung konvergiert μ^(n) für n→∞ garantiert gegen π — unabhängig vom Startzustand?

Irreduzibilität allein reicht nicht — eine periodische Kette oszilliert für immer. Erst irreduzibel + aperiodisch garantiert Konvergenz.

8.🎓 Klausur SS24: Das Gen-Modell hat p_MM = 1. Wie nennt man so einen Zustand?

p_MM = 1 heißt: einmal in M, für immer in M — genau das ist die Definition von absorbierend.

9.🎓 Klausur SS24: Start (1,0,0), erste Zeile von P² ist (0.48, 0.33, 0.19). Welcher Zustand ist bei t=2 am wahrscheinlichsten?

μ^(2) = (0.48, 0.33, 0.19) für (A,I,M) — mit 0.48 ist Aktiv am wahrscheinlichsten.

10.🎓 Klausur SS25: Für P(X₀=V, X₂=V) gilt μ^(0)(V)=0.3 und Σₛ p_Vs·p_sV = 0.44. Wie groß ist P(X₀=V, X₂=V)?

P(X₀=V, X₂=V) = P(X₀=V) · Σₛ p_Vs·p_sV = 0.3 · 0.44 = 0.132.
