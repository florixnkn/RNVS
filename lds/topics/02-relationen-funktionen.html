Worum geht's?

Eine Relation ist einfach eine Menge von Paaren — eine Aussage darüber, welche Elemente irgendwie "in Beziehung" stehen. Eine Funktion ist eine besondere, strengere Relation: jede Eingabe bekommt genau eine Ausgabe. Die Klausur fragt praktisch immer: Welche Eigenschaften hat diese konkrete Relation/Funktion? Dafür brauchst du vier plus drei feste Checklisten — die lernst du hier bildlich.

Stell dir vor …

… du hast zwei Relationen im Alltag: "ist befreundet mit" und "ist Vorgesetzter von".
"Befreundet mit" ist symmetrisch (wenn A mit B befreundet ist, ist B auch mit A befreundet) — aber nicht transitiv (A und B befreundet, B und C befreundet, heißt nicht A und C sind befreundet).
"Vorgesetzter von" ist transitiv (Chef meines Chefs ist auch über mir) und antisymmetrisch (wenn A über B steht UND B über A, dann sind es dieselbe Person) — das ist das Muster einer Ordnung.
Eine Funktion ist dagegen wie ein Postbote: jeder Briefkasten (Eingabe) bekommt genau ein Paket zugeordnet — nie zwei, nie keins.

Relationseigenschaften im Bild

Eine Relation R ⊆ A × A kann vier Eigenschaften haben. Am Beispiel R = {(1,1), (2,2), (3,3), (1,2), (2,3), (1,3)} auf {1,2,3} siehst du alle vier auf einmal:

EigenschaftDefinitionBild-Check
Reflexiv∀x ∈ A: (x,x) ∈ RHat jeder Knoten eine Selbstschleife?
Symmetrisch(x,y)∈R ⇒ (y,x)∈RLäuft zu jedem Pfeil auch einer zurück?
Antisymmetrisch(x,y)∈R ∧ (y,x)∈R ⇒ x=yGibt es (außer Schleifen) keine Pfeile in beide Richtungen?
Transitiv(x,y)∈R ∧ (y,z)∈R ⇒ (x,z)∈RFür jeden 2-Schritt-Pfad gibt es auch die direkte Abkürzung?

Kombinationen ergeben Ordnungstypen

Reflexiv ∧ Symmetrisch ∧ Transitiv
         ↓
    ÄQUIVALENZRELATION        (zerlegt die Menge in Klassen)

Reflexiv ∧ Antisymmetrisch ∧ Transitiv
         ↓
    PARTIELLE ORDNUNG         (Halbordnung — manche Elemente unvergleichbar)

Partielle Ordnung + Total (∀x,y: x≤y ∨ y≤x)
         ↓
    TOTALE ORDNUNG            (linear geordnet, wie eine Kette)
Unser Beispiel R

Reflexiv ✓, Antisymmetrisch ✓ (keine Rückpfeile außer Schleifen), Transitiv ✓ — aber nicht symmetrisch ((1,2)∈R aber (2,1)∉R). Also: Partielle Ordnung.

Funktionen: injektiv, surjektiv, bijektiv

Eine Funktion f: A → B ordnet jedem x ∈ A genau ein y ∈ B zu. Die drei Eigenschaften beantworten je eine eigene Frage:

EigenschaftDefinitionFrage
Injektivf(x₁)=f(x₂) ⇒ x₁=x₂Höchstens ein Urbild pro Wert?
Surjektiv∀y∈B ∃x∈A: f(x)=yWird jedes y getroffen?
BijektivInjektiv ∧ SurjektivEins-zu-eins-Entsprechung?

Permutationen: bijektive Funktionen f: A → A

Eine Permutation bildet eine Menge auf sich selbst ab, bijektiv. In Zyklenschreibweise schreibt man nur, wohin jedes Element wandert, bis der Kreis sich schließt:

Schritt-für-Schritt: Eigenschaften prüfen

Relation prüfen: R = {(1,1),(2,2),(3,3),(1,2),(2,3),(1,3)}

  1. Reflexivität: Sind (1,1), (2,2), (3,3) alle in R? Ja ✓
  2. Symmetrie: Für jedes (x,y) auch (y,x)? (1,2)∈R, aber (2,1)∉R → ✗ nicht symmetrisch
  3. Antisymmetrie: Gibt es (x,y)∧(y,x) mit x≠y? Keine solchen Paare gefunden → ✓ erfüllt (vacuously wahr)
  4. Transitivität: (1,2)&(2,3)→(1,3)? Ja, in R. Alle weiteren Ketten prüfen → ✓
  5. Ergebnis: Reflexiv + Antisymmetrisch + Transitiv → Partielle Ordnung

Funktion prüfen: f: ℤ → ℤ mit f(n) = 2n

  1. Injektiv? f(n₁)=f(n₂) ⇒ 2n₁=2n₂ ⇒ n₁=n₂. Ja ✓
  2. Surjektiv? Gibt es n mit f(n)=3? 2n=3 ⇒ n=1,5 (nicht in ℤ). Nein ✗
  3. Bijektiv? Injektiv ja, surjektiv nein → Nein
  4. Ergebnis: f ist injektiv, aber nicht surjektiv (die ungeraden Zahlen werden nie getroffen)
Merksatz

Prüfe Injektivität algebraisch (Gleichung f(x₁)=f(x₂) auflösen), Surjektivität durch Gegenprobe (kannst du für ein beliebiges y ein passendes x finden — oder eben nicht?).

Typische Fallen

❌ FalschAntisymmetrisch ist einfach "das Gegenteil von symmetrisch".
✅ RichtigAntisymmetrisch: (x,y)∧(y,x)∈R ⇒ x=y. Bei ≤ gilt: 1≤2 aber nicht 2≤1 → das ist völlig ok für Antisymmetrie (nur wenn beide Richtungen gelten, müssen die Elemente gleich sein).
❌ Falsch< ist reflexiv, weil a ≤ a "irgendwie ähnlich" ist.
✅ Richtiga < a ist nie wahr → < ist NICHT reflexiv. ≤ dagegen ist reflexiv (a ≤ a gilt immer).
❌ FalschWenn (1,2) und (2,3) in R sind, dann ist R automatisch transitiv.
✅ RichtigTransitivität muss für alle passenden Paare gelten. Gegenbeispiel: R={(1,2),(2,3)} — (1,2)∧(2,3)∈R, aber (1,3)∉R → NICHT transitiv!
❌ Falsch{(1,2),(2,3)} ist eine Funktion von {1,2,3} nach ℕ.
✅ RichtigJedes Element im Definitionsbereich braucht genau einen Output. Für 3 fehlt ein Bild — das ist keine vollständig definierte Funktion.
❌ FalschInjektiv bedeutet "jedes Element wird getroffen".
✅ RichtigInjektiv = keine zwei verschiedenen x landen auf demselben y. Surjektiv = jedes y wird getroffen. Bijektiv = beides.
❌ FalschWenn R und S beide symmetrisch sind, ist auch R∩S transitiv.
✅ RichtigEigenschaften übertragen sich nicht automatisch auf Kombinationen von Relationen. Jede kombinierte Relation muss einzeln neu geprüft werden.

Quick-Reference

🎯 Quick-Reference für die Klausur

Checkliste: Relationseigenschaften auf Menge A

☐ REFLEXIV:        ∀x: (x,x) ∈ R ?
☐ SYMMETRISCH:      (x,y) ∈ R ⇒ (y,x) ∈ R ?
☐ ANTISYMMETRISCH:  (x,y)∧(y,x) ∈ R ⇒ x=y ?
☐ TRANSITIV:        (x,y)∧(y,z) ∈ R ⇒ (x,z) ∈ R ?

REFL + SYM + TRANS     → ÄQUIVALENZRELATION
REFL + ANTISYM + TRANS → PARTIELLE ORDNUNG (+ TOTAL → TOTALE ORDNUNG)

INJEKTIV?   f(a₁)=f(a₂) ⇒ a₁=a₂ ?          (höchstens 1 Urbild pro Wert)
SURJEKTIV?  ∀b∈B ∃a∈A: f(a)=b ?             (jedes Element von B getroffen)
BIJEKTIV?   Injektiv ∧ Surjektiv ?          (Umkehrfunktion existiert)

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.R = {(1,2),(2,3)} auf {1,2,3}. Ist R transitiv?

(1,2) und (2,3) sind in R, aber (1,3) fehlt → nicht transitiv.

2.Welche Kombination von Eigenschaften ergibt eine Äquivalenzrelation?

Reflexiv + Symmetrisch + Transitiv = Äquivalenzrelation (zerlegt die Menge in Klassen). Reflexiv + Antisymmetrisch + Transitiv wäre eine partielle Ordnung.

3.f: ℤ → ℤ, f(n) = 2n. Ist f surjektiv?

f(n)=3 hätte n=1,5 zur Folge — keine ganze Zahl. Also wird 3 (und jede ungerade Zahl) nie getroffen → nicht surjektiv. Injektiv ist f trotzdem.

4.f(0)=2, f(1)=3, f(2)=1, f(3)=0. Wie lautet die Zyklenschreibweise (Format: "(0 2 1 3)")?

Verfolge die Kette: 0→2→1→3→0, ein einziger Kreis über alle 4 Elemente → (0 2 1 3).

5.f: {1,2} → {1,2,3}, f(1)=1, f(2)=2. Welche Eigenschaft(en) hat f?

1→1 und 2→2 sind verschiedene Bilder → injektiv. Aber 3 ∈ B wird nie getroffen → nicht surjektiv.

6.Die Relation ≤ auf ℤ ist reflexiv, antisymmetrisch, transitiv UND total (alle Zahlen vergleichbar). Wie heißt dieser Ordnungstyp?

Partielle Ordnung + Total (jedes Paar vergleichbar) = totale Ordnung — wie eine durchgehende Kette.
