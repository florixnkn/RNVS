Worum geht's?

Polynome verhalten sich verblüffend wie ganze Zahlen — nur mit x statt Ziffern. Es gibt eine Division mit Rest, einen größten gemeinsamen Teiler (GGT), und sogar "Primzahlen" (irreduzible Polynome). Der einzige Unterschied: Alles findet über einem Körper K statt — und über welchem Körper du rechnest, kann das Ergebnis komplett verändern.

Stell dir vor …

… du kennst den euklidischen Algorithmus für ganze Zahlen: ggT(12, 8) — teile 12 durch 8, Rest 4; teile 8 durch 4, Rest 0; GGT = 4. Für Polynome läuft exakt derselbe Algorithmus, nur dass "teilen mit Rest" jetzt Polynomdivision bedeutet. Und so wie 4 keine kleineren Faktoren außer 1 und sich selbst hat, gibt es Polynome, die sich nicht weiter in Faktoren kleineren Grades zerlegen lassen — die "Primzahlen der Polynomwelt".

Polynome über Körpern

Definition: Ein Polynom über Körper K ist p(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + … + a₁x + a₀, wobei alle Koeffizienten aᵢ ∈ K.

RegelFormel
Grad eines Produktsdeg(p·q) = deg(p) + deg(q)
Grad einer Summedeg(p+q) ≤ max(deg(p), deg(q))
deg(p·q)= deg(p) + deg(q) — addieren!
deg(p+q)≤ max(deg(p), deg(q))
aᵢ ∈ KKoeffizienten müssen im Körper liegen

Polynomdivision

Genau wie bei ganzen Zahlen gibt es Division mit Rest: p(x) = q(x)·d(x) + r(x), mit deg(r) < deg(d).

  1. Teile den höchsten Grad-Term von p durch den höchsten Grad-Term von d (ergibt einen Term des Quotienten).
  2. Multipliziere d mit diesem Term.
  3. Subtrahiere das Ergebnis von p — das ergibt einen neuen (kleineren) Rest.
  4. Wiederhole, bis deg(Rest) < deg(d).
Beispiel: p(x) = x³ + 2x² + 3x + 4 durch q(x) = x + 1
      x² +  x  + 2
    _________________
x+1 | x³ + 2x² + 3x + 4
      x³ +  x²
      -----------
           x² + 3x
           x² +  x
           --------
               2x + 4
               2x + 2
               ------
                    2   ← Rest

Ergebnis: x³+2x²+3x+4 = (x+1)·(x²+x+2) + 2

GGT von Polynomen: Euklidischer Algorithmus

Definition: ggT(p,q) ist das Polynom größten Grades, das sowohl p als auch q teilt. Genau wie bei Zahlen gilt die Bézout-Identität: Es existieren Polynome s,t mit ggT(p,q) = s·p + t·q.

Pseudocode
ggT(p,q): while q≠0: (p,q) ← (q, p mod q); return p
Kurzbeispiel: ggT(x³−1, x²−1)

Schritt 1: x³−1 = (x²−1)·x + (x−1)
Schritt 2: x²−1 = (x−1)·(x+1) + 0
→ ggT(x³−1, x²−1) = x−1

🎓 Klausuraufgabe: GGT über ℤ₅

Klausur SoSe 2025, Aufgabe 5 (12 Punkte). Berechne mit dem euklidischen Algorithmus: ggT(x⁵+x³+x²+3x+2,   x²+3x+2) über dem Körper ℤ₅ (alle Koeffizienten mod 5!).

3⁻¹ = 2 in ℤ₅ (da 3·2=6≡1), also x²/3x → Faktor 2x.

Ergebnis: GGT = 3x + 1

Der letzte Rest ≠ 0 war r₁(x) = 3x+1 — das ist der GGT. Da ℤ₅ ein Körper ist, ist jedes skalare Vielfache ≠ 0 ebenfalls ein gültiger GGT (Assoziierte):

Üblicherweise gibt man die normierte Form an (führender Koeffizient = 1): x + 2.

Irreduzible Polynome

Definition: p ist irreduzibel über K, wenn deg(p) ≥ 1 und p sich nicht als Produkt p = a·b mit deg(a) ≥ 1 und deg(b) ≥ 1 schreiben lässt. Merksatz: irreduzible Polynome sind die "Primzahlen der Polynome".

Faustregel für Grad 2 und 3

Für Polynome vom Grad 2 oder 3 gilt: p hat eine Nullstelle in K ⟺ p ist reduzibel über K. (Achtung: bei Grad ≥ 4 gilt das nicht mehr — ein Polynom kann ohne Nullstelle trotzdem in zwei quadratische Faktoren zerfallen!)

Typische Fallen

❌ Falsch(x³+2x²) / x = x²+2x (ganzer Term auf einmal geteilt).
✅ RichtigNur der höchste Term wird geteilt: x³/x = x². Dann multiplizieren, subtrahieren, weitermachen.
❌ FalschggT(x²−1, x−1) = 1, wie bei "teilerfremden" Zahlen.
✅ RichtigggT(x²−1, x−1) = x−1 — das Polynom selbst! Polynome sind keine Zahlen, "teilerfremd" bedeutet hier ggT = konstantes Polynom ≠ x−1.
❌ Falsch"x²+1 ist überall irreduzibel."
✅ RichtigIrreduzibilität hängt vom Körper ab: über ℝ irreduzibel, über ℂ reduzibel zu (x−i)(x+i), über ℤ₂ reduzibel zu (x+1)². Körper immer angeben!
❌ FalschBei Rechnung über ℤₚ am Ende vergessen, mod p zu reduzieren.
✅ RichtigJeder Koeffizient — auch bei Zwischenschritten — muss mod p reduziert werden, sonst rechnest du in ℤ statt in ℤₚ weiter.

Quick-Reference

🎯 Quick-Reference für die Klausur

Polynomdivision — Algorithmus

1. Teile höchsten Grad-Term von p durch höchsten Grad-Term von q
2. Multipliziere q mit diesem Quotiententerm
3. Subtrahiere das Ergebnis von p
4. Wiederhole, bis deg(Rest) < deg(q)

GGT mit euklidischem Algorithmus

ggT(p, q):
  while q ≠ 0:
    r = p mod q
    p = q
    q = r
  return p   // letzter Nicht-Null-Rest

Irreduzibilitäts-Test

p irreduzibel über K wenn:
☐ deg(p) ≥ 1
☐ keine Faktorisierung p = a·b mit deg(a),deg(b) ≥ 1 in K[x]

Für Grad 2/3: Nullstelle in K? → reduzibel. Keine? → irreduzibel.

Klausur-Ergebnis: ggT über ℤ₅

SchrittDivisionRest

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.Welchen Grad hat p(x) = x⁴ + 3x − 2?

Der höchste Exponent mit Koeffizient ≠ 0 ist 4, also deg(p) = 4.

2.deg(p) = 3, deg(q) = 2. Wie groß ist deg(p·q)?

deg(p·q) = deg(p) + deg(q) = 3 + 2 = 5 (Grade addieren sich, NICHT wie bei der Summe!).

3.Wie lautet ggT(x³−1, x²−1)?

x³−1 = (x²−1)·x + (x−1); x²−1 = (x−1)(x+1) + 0 → GGT = x−1.

4.🎓 Klausur 2025: Welchen Grad hat ggT(x⁵+x³+x²+3x+2, x²+3x+2) über ℤ₅?

Der euklidische Algorithmus liefert GGT = 3x+1 — ein Polynom vom Grad 1.

5.🎓 Klausur 2025: Gib den GGT aus Aufgabe 4 in normierter Form an (führender Koeffizient = 1).

3x+1 mit 3⁻¹=2 (in ℤ₅) multipliziert: 2·(3x+1) = 6x+2 ≡ x+2 (mod 5).

6.Ist x² + 1 irreduzibel über ℝ?

x²+1=0 hätte x=±i zur Lösung — keine reelle Nullstelle, also über ℝ irreduzibel.

7.Ist x² + 1 irreduzibel über ℂ?

Über ℂ gilt x²+1 = (x−i)(x+i) — eine echte Faktorisierung, also reduzibel.
