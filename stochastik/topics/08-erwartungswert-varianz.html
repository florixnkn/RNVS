Worum geht's?

Eine Zufallsvariable X nimmt viele verschiedene Werte an — aber wenn du sie auf eine einzige Zahl zusammenfassen müsstest, welche wäre das? Der Erwartungswert E[X] ist genau diese Zahl: der mit Wahrscheinlichkeiten gewichtete Durchschnitt. Die Varianz Var(X) beantwortet die nächste Frage: Wie stark streuen die Werte typischerweise um diesen Durchschnitt?

Stell dir vor …

… du spielst ein Würfelspiel, bei dem du je nach Wurf 1€, 2€ oder 3€ gewinnst — und zwar mit unterschiedlicher Wahrscheinlichkeit: 1€ mit 20%, 2€ mit 50%, 3€ mit 30%. Spielst du das Spiel sehr oft, pendelt sich dein durchschnittlicher Gewinn pro Runde auf einen festen Wert ein — das ist E[X]. Manchmal gewinnst du deutlich mehr, manchmal weniger; wie stark diese Ausschläge im Schnitt sind, misst Var(X). Ein Spiel mit E[X]=2€ kann trotzdem sehr unterschiedlich "riskant" sein, je nachdem wie breit die Auszahlungen streuen.

Erwartungswert & Varianz im Bild

Der Erwartungswert ist im Kern ein Schwerpunkt: Denk an eine Wippe (Balkenwaage) auf dem Zahlenstrahl, an der bei jedem möglichen Wert x ein Gewicht der Größe P(X=x) hängt. E[X] ist genau der Punkt, an dem die Wippe im Gleichgewicht ist.

Formel: Erwartungswert
Diskret: E[X] = Σ x · P(X=x) Stetig: E[X] = ∫ x · f(x) dx

Existiert nur, wenn diese Summe/dieses Integral absolut konvergiert — bei Verteilungen mit sehr "schweren" Rändern kann E[X] auch undefiniert sein.

Rechenregeln: Linearität

E[X] verhält sich linear — das macht es zum bequemsten Werkzeug der ganzen Stochastik. Diese Regel gilt immer, egal ob X und Y unabhängig sind oder nicht:

RegelGilt …
E[aX + b] = a·E[X] + bimmer
E[X + Y] = E[X] + E[Y]immer — auch bei Abhängigkeit!
E[aX + bY] = a·E[X] + b·E[Y]immer (allgemeine Linearität)
E[X·Y] = E[X]·E[Y]nur bei Unabhängigkeit von X, Y
LOTUS: Erwartungswert einer Funktion von X

Willst du E[g(X)] berechnen (z.B. E[X²]), musst du nicht erst die Verteilung von g(X) herleiten. Du kannst g(x) direkt in die Summe/das Integral einsetzen:

E[g(X)] = Σ g(x)·P(X=x) bzw. E[g(X)] = ∫ g(x)·f(x) dx

Achtung: E[g(X)] ≠ g(E[X]) im Allgemeinen — insbesondere ist E[X²] ≠ (E[X])²!

Varianz: wie stark streut X?

Var(X) misst den mittleren quadrierten Abstand zum Erwartungswert. Quadriert wird, damit sich positive und negative Abweichungen nicht gegenseitig aufheben.

Formel: Varianz & Verschiebungssatz
Definition: Var(X) = E[(X − E[X])²] Verschiebungssatz: Var(X) = E[X²] − (E[X])² ← einfacher zu berechnen! Standardabw.: σ_X = √Var(X)

Der Verschiebungssatz ist rechnerisch fast immer der bequemere Weg: erst E[X] und E[X²] bestimmen (beides über LOTUS), dann subtrahieren.

Eigenschaften der Varianz

RegelBemerkung
Var(aX + b) = a²·Var(X)additive Konstante b fällt weg, a wird quadriert
Var(X + Y) = Var(X) + Var(Y)nur bei Unabhängigkeit von X, Y
Var(X + Y) = Var(X) + Var(Y) + 2·Cov(X,Y)allgemein (siehe Thema 10)
Var(X) ≥ 0immer — Quadrate sind nie negativ
Var(X) = 0 ⟺ X = cX ist dann deterministisch (eine Konstante)
σ = √Var(X)Standardabweichung, gleiche Einheit wie X
E[X²] ≥ (E[X])²Gleichheit nur bei Var(X)=0
Var = 2. Momentzentriert um E[X]
Median0.5-Quantil, P(X≤x₀.₅)=0.5

Schritt-für-Schritt: Beispiele

Beispiel 1: E[X] und Var(X) diskret berechnen

X nimmt die Werte {1,2,3} an mit P(X=1)=0.2, P(X=2)=0.5, P(X=3)=0.3 (das Beispiel von oben).

  1. E[X] berechnen:
    E[X] = 1·0.2 + 2·0.5 + 3·0.3 = 0.2 + 1.0 + 0.9 = 2.1
  2. E[X²] berechnen (LOTUS mit g(x)=x²):
    E[X²] = 1²·0.2 + 2²·0.5 + 3²·0.3 = 0.2 + 2.0 + 2.7 = 4.9
  3. Verschiebungssatz anwenden:
    Var(X) = E[X²] − (E[X])² = 4.9 − 2.1² = 4.9 − 4.41 = 0.49
  4. Standardabweichung:
    σ_X = √0.49 = 0.7
Direkt weiterrechnen mit den Rechenregeln

Für Y = 3X + 2 brauchst du keine neue Rechnung von vorn:
E[Y] = 3·E[X] + 2 = 3·2.1 + 2 = 8.3
Var(Y) = 3²·Var(X) = 9·0.49 = 4.41 (die additive 2 fällt komplett weg!)

Beispiel 2: E[X] stetig berechnen

X hat die Dichte f(x) = 3x² auf [0,1]. Gesucht: E[X].

  1. LOTUS für stetige ZV einsetzen:
    E[X] = ∫₀¹ x · 3x² dx = ∫₀¹ 3x³ dx
  2. Integrieren:
    = 3 · [x⁴/4]₀¹ = 3 · (1/4 − 0) = 3/4 = 0.75

🧮 Klausurtyp: Erwartungswert & Varianz einer Summe unabhängiger Binomialverteilungen

Ein Versandhändler verschickt N = 950 Pakete, davon 570 aufs Land und 380 in die Stadt. Ein Paket vom Land verspätet sich mit Wahrscheinlichkeit 0.08, eines aus der Stadt mit Wahrscheinlichkeit 0.28 — unabhängig voneinander. V = Gesamtzahl verspäteter Pakete. Gesucht: E[V] und Var(V).

  1. Aufteilen: V = V_Land + V_Stadt, mit V_Land ~ B(570, 0.08) und V_Stadt ~ B(380, 0.28), unabhängig.
  2. Kennzahlen der Binomialverteilung einsetzen (E[X]=np, Var(X)=np(1−p)):
    E[V_Land] = 570·0.08 = 45.6  ·  Var(V_Land) = 570·0.08·0.92 = 41.952
    E[V_Stadt] = 380·0.28 = 106.4  ·  Var(V_Stadt) = 380·0.28·0.72 = 76.608
  3. Linearität von E (immer!) und Var (weil unabhängig!) anwenden:
    E[V] = 45.6 + 106.4 = 152
    Var(V) = 41.952 + 76.608 = 118.56
Warum darf man Var(V_Land) + Var(V_Stadt) addieren?

Weil Land- und Stadt-Verspätungen hier unabhängig sind. Ohne diese Annahme müsste man Var(V) = Var(V_Land) + Var(V_Stadt) + 2·Cov(V_Land, V_Stadt) rechnen (siehe Thema 10).

Typische Fallen

❌ FalschVar(X+Y) = Var(X) + Var(Y) — immer.
✅ RichtigDas gilt nur bei Unabhängigkeit. Allgemein: Var(X+Y) = Var(X) + Var(Y) + 2·Cov(X,Y).
❌ FalschVar(3X + 2) = 3·Var(X) + 2
✅ RichtigVar(3X + 2) = 3²·Var(X) = 9·Var(X) — die additive Konstante fällt weg, der Faktor wird quadriert.
❌ FalschE[X²] = (E[X])²
✅ RichtigE[X²] = (E[X])² + Var(X) ≥ (E[X])² — Gleichheit gilt nur, wenn Var(X) = 0 ist (X ist dann konstant).
❌ FalschE[X·Y] = E[X]·E[Y] — für beliebige X, Y.
✅ RichtigDas gilt nur, wenn X und Y unabhängig (oder zumindest unkorreliert) sind.
❌ FalschUm E[3X+5] zu berechnen, muss ich erst die Verteilung von 3X+5 herleiten und neu integrieren.
✅ RichtigLinearität direkt nutzen: E[3X+5] = 3·E[X] + 5 — keine neue Rechnung nötig.

Quick-Reference

🎯 Quick-Reference für die Klausur

Kernformeln

E[X]        = Σ x·P(X=x)                    (diskret)
E[aX+b]     = a·E[X] + b                    (Linearität, immer)
E[X+Y]      = E[X] + E[Y]                   (immer, auch abhängig)
Var(X)      = E[X²] − (E[X])²               (Verschiebungssatz)
Var(aX+b)   = a²·Var(X)                     (b fällt weg, a wird quadriert)
Var(X+Y)    = Var(X)+Var(Y)                 (NUR bei Unabhängigkeit)
σ_X         = √Var(X)                       (Standardabweichung)

Merksätze

  • E ist immer linear — E[X+Y]=E[X]+E[Y] gilt unabhängig von Abhängigkeit
  • Var braucht Unabhängigkeit, um additiv zu sein
  • Bei Var(aX+b): a² · Var(X), b verschwindet komplett
  • Verschiebungssatz statt Definition rechnen: Var(X) = E[X²] − (E[X])²
  • Erst prüfen ob E[X²] endlich ist, bevor du Var(X) berechnest

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.X ist eine faire Münze: P(X=0)=0.5, P(X=1)=0.5 (Bernoulli mit p=0.5). Wie groß ist E[X]?

E[X] = 0·0.5 + 1·0.5 = 0.5.

2.Für dieselbe faire Münze (Bernoulli p=0.5): wie groß ist Var(X)?

Var(X) = p(1−p) = 0.5·0.5 = 0.25. Über den Verschiebungssatz: E[X²] = 0.5, Var(X) = 0.5 − 0.5² = 0.25.

3.Welche Formel für Var(aX+b) ist korrekt?

Var(aX+b) = a²·Var(X) — die additive Konstante b fällt weg, der Faktor a wird quadriert.

4.Gegeben Var(X) = 4. Wie groß ist Var(3X + 5)?

Var(3X+5) = 3²·Var(X) = 9·4 = 36.

5.Gilt Var(X+Y) = Var(X) + Var(Y) für beliebige X und Y (ja/nein)?

Nein — nur bei Unabhängigkeit. Allgemein gilt Var(X+Y) = Var(X) + Var(Y) + 2·Cov(X,Y).

6.🧮 Klausurtyp: X∈{1,2,3} mit P=0.2, 0.5, 0.3 (E[X]=2.1, Var(X)=0.49, wie im Beispiel oben). Wie groß ist die Standardabweichung σ_X?

σ_X = √Var(X) = √0.49 = 0.7.

7.🧮 Klausurtyp: Wann gilt E[X·Y] = E[X]·E[Y]?

E[XY] = E[X]·E[Y] gilt nur bei Unabhängigkeit (bzw. Unkorreliertheit) von X und Y — im Gegensatz zu E[X+Y]=E[X]+E[Y], das immer gilt.
