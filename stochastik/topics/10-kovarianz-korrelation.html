Worum geht's?

Bisher hast du einzelne Zufallsvariablen betrachtet. Jetzt geht es um zwei gleichzeitig: Hängen X und Y zusammen? Steigt Y tendenziell, wenn X steigt (oder fällt)? Die Kovarianz Cov(X,Y) misst diesen Zusammenhang als Zahl, der Korrelationskoeffizient ρ normiert sie auf eine leicht interpretierbare Skala zwischen −1 und 1.

Stell dir vor …

… jemand zeigt dir eine Statistik: An Tagen mit mehr Eisverkäufen ertrinken auch mehr Menschen. Bedeutet das, Eis essen sei gefährlich? Nein — beides hängt von einer dritten Größe ab, der Temperatur: An heißen Tagen wird mehr Eis gegessen und mehr geschwommen (und damit steigt auch das Ertrinkungsrisiko). Eisverkäufe und Ertrinken sind korreliert, aber keins verursacht das andere — das ist der Kern von "Korrelation ≠ Kausalität", einer der wichtigsten Lehren dieses Themas.

Kovarianz & Korrelation im Bild

Die Kovarianz vergleicht für jeden Messpunkt, ob X und Y gemeinsam über oder unter ihrem jeweiligen Mittelwert liegen. Liegen beide meistens auf derselben Seite (beide über oder beide unter dem Mittel), ist Cov(X,Y) positiv; liegen sie meist auf entgegengesetzten Seiten, ist sie negativ.

Formel: Kovarianz
Cov(X,Y) = E[(X − E[X])(Y − E[Y])] = E[XY] − E[X]·E[Y] ← Rechenschema (Verschiebungssatz)

Cov(X,Y) > 0: X und Y steigen/fallen tendenziell zusammen. Cov(X,Y) < 0: wenn X steigt, fällt Y tendenziell. Cov(X,Y) = 0: X und Y sind unkorreliert.

Eigenschaften der Kovarianz

RegelBemerkung
Cov(X,X) = Var(X)Kovarianz mit sich selbst ist die Varianz
Cov(X,Y) = Cov(Y,X)Symmetrie
Cov(aX+b, cY+d) = a·c·Cov(X,Y)bilinear — additive Konstanten fallen weg
Cov(X, Y+Z) = Cov(X,Y) + Cov(X,Z)Bilinearität in beiden Argumenten
X, Y unabhängig ⟹ Cov(X,Y) = 0aber nicht umgekehrt!

Korrelationskoeffizient: die normierte Version

Die Kovarianz hängt von den Einheiten von X und Y ab (z.B. cm vs. m) — das macht sie schwer vergleichbar. Der Korrelationskoeffizient ρ normiert die Kovarianz auf das Intervall [−1,1]:

Formel: Korrelationskoeffizient (Pearson)
ρ_{XY} = Cor(X,Y) = Cov(X,Y) / (σ_X · σ_Y) −1 ≤ ρ ≤ 1 immer! ρ = 1 ⟺ Y = aX + b mit a > 0 perfekte positive lineare Abhängigkeit ρ = −1 ⟺ Y = aX + b mit a < 0 perfekte negative lineare Abhängigkeit ρ = 0 ⟺ X, Y unkorreliert kein linearer Zusammenhang

Varianz einer Summe (allgemein)

Jetzt lässt sich die Formel aus Thema 08 vervollständigen — ohne Unabhängigkeitsannahme kommt ein Kovarianz-Term hinzu:

Formel: Var(X+Y) allgemein
Var(X + Y) = Var(X) + Var(Y) + 2·Cov(X,Y) Allgemein: Var(Σᵢ Xᵢ) = Σᵢ Var(Xᵢ) + 2·Σ_{i<j} Cov(Xᵢ,Xⱼ) Bei Unabhängigkeit fallen alle Kovarianzen weg: Var(Σᵢ Xᵢ) = Σᵢ Var(Xᵢ)

Achtung: unkorreliert ≠ unabhängig

Cov(X,Y) = 0 bedeutet nur, dass es keinen linearen Zusammenhang gibt. Ein nicht-linearer (aber sehr starker!) Zusammenhang kann trotzdem bestehen:

Sonderfall: bivariate Normalverteilung

Bei (X,Y) ~ N₂(μ_X, μ_Y, σ_X², σ_Y², ρ) gilt ausnahmsweise: ρ = 0 ⟺ X, Y unabhängig. Das ist die einzige gängige Verteilung, bei der Unkorreliertheit automatisch Unabhängigkeit bedeutet — verlass dich darauf nicht bei anderen Verteilungen!

Cov(X,X) = Var(X)Sonderfall der Kovarianz
ρ ∈ [−1,1]dimensionslos, dank Normierung
Unabh. ⟹ Cov=0Rückrichtung gilt i.A. nicht
Σᵢⱼ = Cov(Xᵢ,Xⱼ)Kovarianzmatrix, symmetrisch & pos. semidefinit

Schritt-für-Schritt: Beispiele

Beispiel 1: Kovarianz und Korrelation aus gemeinsamer Verteilung

X und Y haben die gemeinsame Wahrscheinlichkeitsfunktion:

f(x,y)y=1y=2
x=10.10.3
x=20.40.2
  1. Randverteilungen bestimmen:
    P(X=1) = 0.4, P(X=2) = 0.6  ·  P(Y=1) = 0.5, P(Y=2) = 0.5
  2. E[X] und E[Y]:
    E[X] = 1·0.4 + 2·0.6 = 1.6  ·  E[Y] = 1·0.5 + 2·0.5 = 1.5
  3. E[XY]:
    E[XY] = 1·1·0.1 + 1·2·0.3 + 2·1·0.4 + 2·2·0.2 = 0.1+0.6+0.8+0.8 = 2.3
  4. Kovarianz:
    Cov(X,Y) = E[XY] − E[X]·E[Y] = 2.3 − 1.6·1.5 = 2.3 − 2.4 = −0.1
  5. Für ρ zusätzlich Var(X) und Var(Y):
    E[X²] = 1·0.4 + 4·0.6 = 2.8 → Var(X) = 2.8 − 1.6² = 0.24 → σ_X ≈ 0.49
    E[Y²] = 1·0.5 + 4·0.5 = 2.5 → Var(Y) = 2.5 − 1.5² = 0.25 → σ_Y = 0.5
  6. Korrelationskoeffizient:
    ρ = Cov(X,Y)/(σ_X·σ_Y) = −0.1/(0.49·0.5) = −0.1/0.245 ≈ −0.41
Plausibilitätscheck

ρ ≈ −0.41 liegt sauber in [−1,1] — ein Wert außerhalb dieses Bereichs wäre ein sicheres Zeichen für einen Rechenfehler bei Cov, Var(X) oder Var(Y).

🧮 Klausurtyp: Kovarianz aus einer 3×3-Kontingenztafel

Gegeben die gemeinsame Verteilung von X und Y:

Y=0Y=1Y=2
X=00.10.20.1
X=10.20.10.1
X=20.10.00.1
  1. Randverteilungen:
    P(X=0)=0.4, P(X=1)=0.4, P(X=2)=0.2  ·  P(Y=0)=0.4, P(Y=1)=0.3, P(Y=2)=0.3
  2. Erwartungswerte:
    E[X] = 0·0.4+1·0.4+2·0.2 = 0.8  ·  E[Y] = 0·0.4+1·0.3+2·0.3 = 0.9
  3. E[XY] — nur Zellen mit x,y ≠ 0 zählen:
    E[XY] = 1·1·0.1 + 1·2·0.1 + 2·1·0.0 + 2·2·0.1 = 0.1+0.2+0+0.4 = 0.7
  4. Kovarianz:
    Cov(X,Y) = 0.7 − 0.8·0.9 = 0.7 − 0.72 = −0.02

Die Kovarianz ist hier sehr klein, aber (leicht) negativ von null verschieden — X und Y sind also schwach negativ korreliert und damit nicht unabhängig.

Typische Fallen

❌ FalschCov(X,Y) = 0 bedeutet, dass X und Y unabhängig sind.
✅ RichtigUnkorreliertheit ≠ Unabhängigkeit (außer bei bivariater Normalverteilung). Gegenbeispiel: X=sin(U), Y=cos(U) mit U~U(0,2π): Cov=0, aber X²+Y²=1 — total abhängig!
❌ FalschIch habe ρ = 1.3 berechnet — kann so stehen bleiben, ist halt eine starke Korrelation.
✅ Richtigρ muss immer in [−1,1] liegen. Ein Wert außerhalb ist ein Rechenfehler — such ihn bei Cov, Var(X) oder Var(Y).
❌ FalschCov(aX+b, Y) = a·Cov(X,Y) + b·Cov(1,Y)
✅ RichtigCov(aX+b, Y) = a·Cov(X,Y) — additive Konstanten fallen bei der Kovarianz immer komplett weg.
❌ FalschVar(X−Y) = Var(X) − Var(Y)
✅ RichtigVar(X−Y) = Var(X) + Var(Y) − 2·Cov(X,Y) — das Vorzeichen vor Cov dreht sich um, aber Var(Y) wird addiert (Varianzen sind nie negativ).
❌ FalschZwei Größen sind korreliert, also verursacht die eine die andere.
✅ RichtigKorrelation ≠ Kausalität. Ein Zusammenhang kann auch durch eine gemeinsame dritte Ursache entstehen (siehe Eisverkauf und Ertrinken — beides hängt von der Temperatur ab).

Quick-Reference

🎯 Quick-Reference für die Klausur

Kernformeln

Cov(X,Y)  = E[XY] − E[X]·E[Y]                Rechenformel
ρ         = Cov(X,Y) / (σ_X · σ_Y)           Pearson-Korrelation, ∈ [−1,1]
Cov(X,X)  = Var(X)                           Sonderfall
Cov(aX+b, cY+d) = a·c·Cov(X,Y)               bilinear, Konstanten fallen weg
Var(X+Y)  = Var(X) + Var(Y) + 2·Cov(X,Y)     allgemeine Varianz der Summe
Unabhängig ⟹ Cov=0                           aber nicht umgekehrt!

Aufgabentypen im Überblick

AufgabentypVorgehen
Kovarianz berechnenE[XY] − E[X]·E[Y] aus gemeinsamer Verteilung
Korrelation bestimmenCov(X,Y) / (σ_X·σ_Y)
Varianz einer LinearkombinationVar(aX+bY) = a²Var(X)+b²Var(Y)+2ab·Cov(X,Y)
Unabhängigkeit vs. UnkorreliertheitCov=0 prüfen reicht für Unabhängigkeit NICHT
Bivariate Normalverteilunghier gilt ausnahmsweise ρ=0 ⟺ unabhängig

Merksätze

  • Cov(X,X) = Var(X) — die Kovarianz verallgemeinert die Varianz
  • ρ ist immer in [−1,1] — Wert außerhalb = Rechenfehler
  • Additive Konstanten fallen bei Cov immer weg, multiplikative werden multipliziert
  • Unabhängig ⟹ unkorreliert, aber die Rückrichtung gilt i.A. nicht
  • Korrelation ist kein Kausalitätsbeweis

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.Für das Beispiel oben gilt E[XY]=2.3, E[X]=1.6, E[Y]=1.5. Wie groß ist Cov(X,Y)?

Cov(X,Y) = 2.3 − 1.6·1.5 = 2.3 − 2.4 = −0.1.

2.Mit Cov(X,Y)=−0.1, σ_X≈0.49, σ_Y=0.5: wie groß ist der Korrelationskoeffizient ρ?

ρ = −0.1/(0.49·0.5) = −0.1/0.245 ≈ −0.41.

3.Welcher Wertebereich gilt immer für den Korrelationskoeffizienten ρ?

ρ liegt immer in [−1, 1] — das folgt aus der Cauchy-Schwarz-Ungleichung.

4.Kann man aus Cov(X,Y)=0 allgemein schließen, dass X und Y unabhängig sind (ja/nein)?

Nein — Unkorreliertheit ist schwächer als Unabhängigkeit (außer im Sonderfall der bivariaten Normalverteilung).

5.Cov(X,Y) = −0.1. Wie groß ist Cov(2X, 3Y)?

Bilinearität: Cov(2X,3Y) = 2·3·Cov(X,Y) = 6·(−0.1) = −0.6.

6.🧮 Klausurtyp: Welche Formel für Var(X−Y) ist korrekt?

Var(X−Y) = Var(X) + (−1)²Var(Y) + 2·(−1)·Cov(X,Y) = Var(X) + Var(Y) − 2·Cov(X,Y).

7.🧮 Klausurtyp: Für die 3×3-Kontingenztafel oben gilt E[XY]=0.7, E[X]=0.8, E[Y]=0.9. Wie groß ist Cov(X,Y)?

Cov(X,Y) = 0.7 − 0.8·0.9 = 0.7 − 0.72 = −0.02.

8.🎓 Klausur SS24: Gemeinsame Verteilung mit E(X) = −0.05, E(Y) = 1.5, E(XY) = 0.1. Was ist Cov(X,Y)?

Cov(X,Y) = E(XY) − E(X)E(Y) = 0.1 − (−0.05·1.5) = 0.1 + 0.075 = 0.175.

9.🎓 Nachklausur SS24: Kontingenztabelle mit E[XY] = −0.1 + θ, E[X]E[Y] = 0.21. Für welches θ sind X und Y unkorreliert?

Unkorreliert bedeutet Cov(X,Y) = 0, also E[XY] = E[X]E[Y]: −0.1 + θ = 0.21 → θ = 0.31. Achtung: unkorreliert heißt hier trotzdem NICHT unabhängig (in der Klausur die Folgefrage!).
