Worum geht's?

Bei diskreten Zufallsvariablen war p(x) direkt eine Wahrscheinlichkeit. Bei stetigen Zufallsvariablen — Größen wie Zeit, Gewicht oder Länge, die jeden Wert in einem Intervall annehmen können — funktioniert das nicht mehr: Es gibt unendlich viele mögliche Werte, jeder einzelne hat Wahrscheinlichkeit 0. Statt einer WK-Funktion arbeitest du jetzt mit einer Dichte — und die ist etwas grundlegend anderes.

Stell dir vor …

… eine Bevölkerungsdichte-Karte: "12.000 Einwohner pro km²" in der Innenstadt. Diese Zahl ist riesig — viel größer als 1 — aber sie verletzt keine Regel, denn sie ist keine "Wahrscheinlichkeit, dass da jemand wohnt", sondern eine Dichte: Erst wenn du sie mit einer Fläche multiplizierst (bzw. integrierst), bekommst du eine echte Anzahl Menschen. Genauso ist es mit der Dichtefunktion f(x) einer stetigen Zufallsvariable: f(x) selbst ist keine Wahrscheinlichkeit und darf über 1 liegen. Erst die Fläche unter der Kurve — das Integral über ein Intervall — ergibt eine echte Wahrscheinlichkeit.

Dichte f(x) & Verteilungsfunktion F(x)

X heißt stetig verteilt, wenn es eine Funktion f_X: ℝ → [0,∞) gibt, die Dichtefunktion, sodass für jede messbare Menge B gilt: P(X ∈ B) = ∫B f_X(x) dx. Für ein Intervall bedeutet das ganz konkret: die Wahrscheinlichkeit ist die Fläche unter der Kurve von f_X.

Eigenschaften der Dichte
(1) f_X(x) ≥ 0 für alle x  ·  (2) ∫−∞∞ f_X(x) dx = 1

Wichtig: P(X = x) = 0 für jeden einzelnen Punkt x — und f_X(x) darf größer als 1 sein! f_X ist eine Dichte, keine Wahrscheinlichkeit.

Verteilungsfunktion: das Integral von −∞ bis x

Formel: Verteilungsfunktion
F_X(x) = P(X ≤ x) = ∫−∞x f_X(t) dt  ·  f_X(x) = F_X'(x)

Anders als bei diskreten ZV ist F_X hier stetig (kein Sprung!), monoton nicht-fallend, mit F_X(x)→0 für x→−∞ und F_X(x)→1 für x→+∞. Ableiten und Integrieren sind zueinander invers: F ist das Integral von f, f ist die Ableitung von F.

P(a≤X≤b) = P(a<X<b)alle Varianten gleich (Punkte zählen nicht)
= F(b) − F(a)egal ob mit oder ohne Rand
Quantil x_pF(x_p) = p, also x_p = F⁻¹(p)
Medianx_{0.5}, das 50%-Quantil

Gleichverteilung U(a,b) — die einfachste stetige Verteilung

Alle Werte in [a,b] sind gleich wahrscheinlich — die Dichte ist ein flaches Rechteck:

Formel: Gleichverteilung
f(x) = 1/(b−a) auf [a,b]  ·  E[X] = (a+b)/2  ·  Var(X) = (b−a)²/12

Schritt-für-Schritt: Gleichverteilung komplett durchrechnen

X ~ U(2, 6). Gesucht: die Dichte, die Verteilungsfunktion, und P(3 ≤ X ≤ 5).

  1. Dichte aufstellen: f(x) = 1/(6−2) = 1/4 für x ∈ [2,6], sonst 0
  2. Verteilungsfunktion integrieren: F(x) = 0 für x<2; F(x) = (x−2)/4 für 2≤x≤6; F(x) = 1 für x>6
  3. P(3≤X≤5) über die Dichte: ∫35 (1/4) dx = (1/4)·(5−3) = 2/4 = 0.5
  4. Gegenprobe über F: F(5)−F(3) = (5−2)/4 − (3−2)/4 = 3/4 − 1/4 = 2/4 = 0.5 ✓

Normierungskonstante bestimmen

f(x) = c·x² für x ∈ [0,1], sonst 0. Bestimme c so, dass f eine gültige Dichte ist.

  1. Bedingung aufstellen: ∫01 c·x² dx = 1
  2. Integrieren: c · [x³/3]01 = c · 1/3 = 1
  3. Auflösen: c = 3

Verteilungsfunktion aus der Dichte bestimmen

Mit c=3 ist f(x) = 3x² für x ∈ [0,1]. Bestimme F(x).

  1. Integrieren: F(x) = ∫0x 3t² dt = [t³]0x = x³ für x ∈ [0,1]
  2. Vollständig hinschreiben: F(x)=0 für x<0; F(x)=x³ für 0≤x≤1; F(x)=1 für x>1
  3. Beispielwert: F(0.5) = 0.5³ = 0.125

Quantil berechnen: F umkehren statt einsetzen

Mit F(x)=x³ auf [0,1]: gesucht ist der Median x0.5, also die Stelle, an der genau die Hälfte der Fläche links liegt.

  1. Quantilgleichung aufstellen: F(x0.5) = x0.5³ = 0.5
  2. Nach x auflösen (dritte Wurzel ziehen): x0.5 = 0.51/3
  3. Auswerten: x0.5 ≈ 0.794
  4. Plausibilitäts-Check: Bei f(x)=3x² ist mehr Masse in der Nähe von x=1 (die Dichte wächst dort an) — deshalb liegt der Median mit ≈0.794 über der "geometrischen Mitte" 0.5 des Intervalls. Bei der Gleichverteilung dagegen ist der Median immer exakt (a+b)/2, weil die Dichte dort konstant ist.

Typische Fallen

❌ FalschBei f(x)=3x² ist P(X=0.5) = f(0.5) = 3·0.25 = 0.75.
✅ RichtigP(X=0.5) = 0 — bei einer stetigen ZV hat jeder einzelne Punkt Wahrscheinlichkeit 0. f(x) ist eine Dichte, keine WK. Nur Integrale (Flächen) geben Wahrscheinlichkeiten!
❌ Falsch"f(x)=5 ist ungültig, eine Wahrscheinlichkeit kann nicht größer als 1 sein."
✅ Richtigf(x)=5 auf [0, 0.2] ist völlig gültig! Das Integral (die Fläche) ergibt 5·0.2 = 1 ✓. Nur die Fläche muss 1 sein, nicht die Höhe.
❌ FalschBei stetigen ZV ist P(a<X<b) etwas anderes als P(a≤X≤b).
✅ RichtigBeide sind gleich! Da einzelne Punkte Wahrscheinlichkeit 0 haben, ändert es nichts, ob die Ränder mitgezählt werden oder nicht.
❌ FalschF(x) = ∫ax f(t) dt reicht als vollständige Antwort, egal für welches x.
✅ RichtigF muss stückweise angegeben werden: 0 für x unterhalb des Trägers, das Integral innerhalb, 1 für x oberhalb — sonst fehlt die vollständige Definition auf ganz ℝ.

Merksätze

  • f(x) ist eine Dichte, keine WK — sie darf > 1 sein
  • P(X=x) = 0 für jeden Einzelpunkt bei stetigen ZV
  • Nur die Fläche zählt — die Höhe von f(x) allein sagt nichts
  • ≤ und < sind bei stetigen ZV austauschbar
  • F immer stückweise vollständig hinschreiben (vor / im / nach dem Träger)

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.X ~ U(2, 6). Wie groß ist P(3 ≤ X ≤ 5)?

f(x)=1/4 auf [2,6]. (1/4)·(5−3) = 0.5.

2.f(x) = c·x² für x ∈ [0,1], sonst 0. Wie muss c gewählt werden, damit f eine gültige Dichte ist?

∫₀¹ c·x² dx = c/3 = 1 → c = 3.

3.Mit f(x)=3x² auf [0,1] ist F(x)=x³. Wie groß ist F(0.5)?

F(0.5) = 0.5³ = 0.125.

4.X ist stetig verteilt. Wie groß ist P(X = 3)?

Bei stetigen ZV hat jeder Einzelpunkt Wahrscheinlichkeit 0 — egal wie groß f(3) ist.

5.f(x)=5 auf [0, 0.2], sonst 0. Ist f eine gültige Dichte?

∫₀^0.2 5 dx = 5·0.2 = 1 ✓. Die Höhe f(x) darf > 1 sein, solange die Fläche 1 ergibt.

6.Wie heißt f_X(x) bei einer stetigen ZV — die Funktion, die selbst keine Wahrscheinlichkeit ist, aber über ein Integral eine WK ergibt?

Die Dichtefunktion (Dichte, pdf).

7.🧮 Klausurtyp: X ~ U(2, 6). Wie groß ist der Median (das 0.5-Quantil)?

F(m) = (m−2)/4 = 0.5 → m = 4. Bei der Gleichverteilung ist der Median immer der Mittelpunkt (a+b)/2.
