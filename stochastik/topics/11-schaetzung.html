Worum geht's?

Du kennst die ganze Grundgesamtheit so gut wie nie — du hast nur eine Stichprobe. Statistische Schätzung ist die Kunst, aus dieser Stichprobe eine möglichst gute Vermutung über einen unbekannten Parameter der Grundgesamtheit (z. B. einen Mittelwert μ oder eine Wahrscheinlichkeit p) abzuleiten — und zusätzlich ehrlich zu sagen, wie sicher diese Vermutung ist.

Stell dir vor …

… du willst wissen, wie viele Gummibärchen in einem riesigen Glas stecken, ohne sie zu zählen. Du nimmst eine Handvoll (Stichprobe), zählst die Farben und rechnest hoch. Das Ergebnis ist dein Punktschätzer — eine einzelne Zahl. Aber du weißt: Bei einer anderen Handvoll käme vielleicht eine leicht andere Zahl raus. Deshalb gibst du zusätzlich einen Bereich an, in dem die wahre Anzahl mit hoher Sicherheit liegt — das Konfidenzintervall. Die Maximum-Likelihood-Methode ist dabei die Faustregel, mit der du aus der Handvoll die plausibelste Gesamtzusammensetzung ableitest.

Schätzer, Maximum-Likelihood & Konfidenzintervalle im Bild

Ein Schätzer θ̂ = T(X₁,…,Xₙ) ist einfach eine Funktion deiner Daten, die eine Zahl ausspuckt. Damit er etwas taugt, brauchst du zwei Gütekriterien:

Gütekriterien
Erwartungstreue: E[θ̂] = θ (im Schnitt richtig) Bias: Bias(θ̂) = E[θ̂] − θ MSE: MSE(θ̂) = Var(θ̂) + Bias(θ̂)²

Ein Schätzer kann im Mittel richtig, aber trotzdem instabil sein (hohe Varianz) — oder stabil, aber systematisch daneben (Bias). Beides zusammen zeigt der MSE.

Die Top-Fehlerquelle: 1/n vs. 1/(n−1)

Die MLE-Varianz σ̂²_MLE = (1/n)·Σ(Xᵢ−X̄)² ist nicht erwartungstreu — sie unterschätzt σ² systematisch. Erwartungstreu ist erst die Stichprobenvarianz S² = (1/(n−1))·Σ(Xᵢ−X̄)², weil ein Freiheitsgrad durch das Schätzen von X̄ "verbraucht" wird.

Kleines Beispiel: Stichprobe {2, 4, 6}, n = 3, X̄ = 4. Σ(Xᵢ−X̄)² = 4+0+4 = 8.
MLE-Variante: 8/3 ≈ 2.667 — Stichprobenvarianz: 8/2 = 4. Zwei unterschiedliche Zahlen aus denselben Daten, je nachdem, welchen Nenner du nimmst!

Maximum-Likelihood-Methode: das 4-Schritte-Rezept

Die Idee: Wähle den Parameter θ, unter dem die beobachteten Daten am wahrscheinlichsten gewesen wären. Dazu maximierst du die Likelihood-Funktion L(θ) — in der Praxis über den Umweg des Logarithmus, weil Summen einfacher abzuleiten sind als Produkte.

ModellMLE-Schätzer
N(μ, σ²)μ̂ = X̄ₙ,   σ̂² = (1/n)·Σ(Xᵢ−X̄)²
Ber(p)p̂ = X̄ₙ = k/n  (Trefferquote)
Poi(λ)λ̂ = X̄ₙ
Exp(λ)λ̂ = 1/X̄ₙ

Konfidenzintervalle: der ehrliche Unsicherheitsbereich

Ein Punktschätzer ist nur eine Zahl — ein (1−α)-Konfidenzintervall [L, U] gibt zusätzlich an, wie unsicher diese Zahl ist. Bei bekanntem σ gilt für den Mittelwert:

Formel: Konfidenzintervall für μ (σ bekannt)
X̄ₙ ± z_{α/2} · σ/√n

Bei unbekanntem σ ersetzt du z_{α/2} durch t_{α/2, n−1} (t-Verteilung) und σ durch die Stichprobenstandardabweichung S.

Niveauz_{α/2}
90%1.645
95%1.96
99%2.576

Schritt-für-Schritt: Beispiele durchgerechnet

Durchgerechnetes Beispiel: MLE für eine Trefferquote (Bernoulli)

Ein Basketballspieler wirft n = 10 Freiwürfe, davon k = 7 Treffer. Wie lautet der Maximum-Likelihood- Schätzer für die Trefferwahrscheinlichkeit p?

  1. Likelihood aufstellen: Jeder Wurf ist Bernoulli-verteilt (Treffer = 1 mit WK p). Für k Treffer aus n Würfen: L(p) = p^k · (1−p)^(n−k)
  2. Logarithmieren: ℓ(p) = k·log(p) + (n−k)·log(1−p)
  3. Ableiten: dℓ/dp = k/p − (n−k)/(1−p)
  4. Null setzen & auflösen: k/p = (n−k)/(1−p) ⟹ k(1−p) = (n−k)p ⟹ k = np ⟹ p̂ = k/n
Eingesetzt

p̂ = k/n = 7/10 = 0.7 — der MLE-Schätzer ist hier genau die beobachtete Trefferquote. Das ist kein Zufall: Für Bernoulli/Binomial ist p̂ = X̄ₙ immer die relative Häufigkeit.

Durchgerechnetes Beispiel: 95%-Konfidenzintervall

Eine Stichprobe von n = 100 Messungen liefert X̄ = 50 bei bekanntem σ = 10. Gesucht: 95%-KI für μ.

  1. Passenden z-Wert wählen: Für 95% ist z_{0.025} = 1.96
  2. Standardfehler berechnen: σ/√n = 10/√100 = 10/10 = 1
  3. Halbbreite berechnen: z_{α/2} · σ/√n = 1.96 · 1 = 1.96
  4. Intervall bilden: [50 − 1.96,   50 + 1.96] = [48.04, 51.96]
🧮 Klausurtyp: Stichprobengröße bestimmen

Häufige Variante: "Wie groß muss n sein, damit das 95%-KI höchstens Breite 2 hat, bei σ = 5?" Halbbreite ≤ 1: 1.96·5/√n ≤ 1 ⟹ √n ≥ 9.8 ⟹ n ≥ 96.04 ⟹ n ≥ 97 (immer aufrunden, sonst wird das Intervall zu breit!).

Typische Fallen

❌ FalschStichprobenvarianz = (1/n)·Σ(Xᵢ−X̄)²
✅ RichtigErwartungstreue Stichprobenvarianz: S² = (1/(n−1))·Σ(Xᵢ−X̄)². Die (1/n)-Variante ist der MLE-Schätzer, aber nicht erwartungstreu.
❌ Falsch"Mit 95% Wahrscheinlichkeit liegt μ im berechneten Intervall."
✅ Richtigμ ist eine feste (wenn auch unbekannte) Zahl — das Intervall ist zufällig. Richtig: "95% aller so konstruierten Intervalle enthalten den wahren Wert μ."
❌ FalschFür ein Konfidenzintervall immer z_{α/2} verwenden.
✅ Richtigσ bekannt → z_{α/2}. σ unbekannt → t_{α/2, n−1} (t-Verteilung, breiter als die Normalverteilung).
❌ FalschDer Maximum-Likelihood-Schätzer ist immer optimal — auch für σ².
✅ Richtigσ̂²_MLE = (1/n)·Σ(Xᵢ−X̄)² ist verzerrt (unterschätzt σ² im Schnitt). MLE-Optimalität bezieht sich auf andere Eigenschaften (z. B. Konsistenz), nicht automatisch auf Erwartungstreue.

SituationFormel

z-Werte für Konfidenzintervalle

90%-KI  →  z = 1.645
95%-KI  →  z = 1.96
99%-KI  →  z = 2.576

Merksätze

  • 1/(n−1) für erwartungstreue Varianz, 1/n für den MLE-Schätzer
  • MLE-Rezept: L(θ) → log → ableiten → =0 setzen — immer dieselben vier Schritte
  • Konfidenzintervall-Interpretation: nicht "P(μ im Intervall)", sondern "95% aller Intervalle treffen μ"
  • σ bekannt → z; σ unbekannt → t (mit n−1 Freiheitsgraden)
  • Bei Ber(p)/Binomial ist p̂ immer einfach die relative Häufigkeit

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.Bei n = 10 Freiwürfen gelingen k = 7 Treffer. Wie lautet der MLE-Schätzer p̂ für die Trefferwahrscheinlichkeit?

p̂ = k/n = 7/10 = 0.7 — die MLE-Rechnung liefert bei Bernoulli/Binomial immer die relative Häufigkeit.

2.Welcher Nenner macht die Stichprobenvarianz erwartungstreu?

S² = (1/(n−1))·Σ(Xᵢ−X̄)² ist erwartungstreu, weil ein Freiheitsgrad durch die Schätzung von X̄ verloren geht. Mit 1/n bekommst du den (verzerrten) MLE-Schätzer.

3.Stichprobe {2, 4, 6}: Wie groß ist die MLE-Varianz σ̂² = (1/n)·Σ(Xᵢ−X̄)²?

X̄ = 4, Σ(Xᵢ−X̄)² = 4+0+4 = 8. σ̂² = 8/3 ≈ 2.667.

4.Wie heißt das Verfahren, das genau den Parameter θ wählt, unter dem die beobachteten Daten am wahrscheinlichsten gewesen wären?

Die Maximum-Likelihood-Methode (MLE) — Likelihood aufstellen, logarithmieren, ableiten, gleich null setzen.

5.🧮 Klausurtyp: n=100, X̄=50, σ=10, 95%-KI für μ. Wie lautet die obere Grenze?

50 + 1.96·(10/√100) = 50 + 1.96 = 51.96.

6.Gleiches Beispiel: Wie lautet die untere Grenze des 95%-KI?

50 − 1.96·(10/√100) = 50 − 1.96 = 48.04.

7.Was bedeutet ein 95%-Konfidenzintervall korrekt interpretiert?

μ ist fest, nicht zufällig — das Intervall ist die Zufallsgröße. Bei hypothetischer Wiederholung würden 95% aller Intervalle μ enthalten.
