Worum geht's?

Du hast Punkte (x,y) — zum Beispiel Lernzeit und Klausurnote, oder Werbebudget und Umsatz — und willst wissen: Wie hängen sie zusammen? Lineare Regression legt die am besten passende Gerade durch deine Punktwolke und liefert dir eine Formel, mit der du für neue x-Werte einen y-Wert vorhersagen kannst.

Stell dir vor …

… du zeichnest Punkte "Berufserfahrung in Jahren" gegen "Gehalt" auf ein Blatt Papier. Die Punkte streuen, aber ein Trend ist sichtbar: mehr Erfahrung, tendenziell mehr Gehalt. Jetzt legst du ein Lineal so an, dass es "im Mittel" möglichst nah an allen Punkten vorbeiläuft. Genau das macht die Kleinste-Quadrate-Methode — nur nicht nach Augenmaß, sondern durch Minimierung der quadrierten senkrechten Abstände (Residuen) zwischen Punkten und Gerade.

KQ-Schätzer & R² im Bild

Das lineare Modell (einfache Regression)
Yᵢ = β₀ + β₁·xᵢ + Uᵢ i = 1,...,n xᵢ: feste Kovariable Yᵢ: zufällige Antwort Uᵢ: Störgröße (Fehler)

Mit mehreren Kovariablen x₁,…,x_k schreibt man das kompakt in Matrixform:

Matrixnotation (allgemeine Regression)
Y = X·β + U Y∈ℝⁿ, X∈ℝ^{n×(k+1)}, β∈ℝ^{k+1}, U∈ℝⁿ β̂ = (XᵀX)⁻¹XᵀY Ŷ = X·β̂ = H·Y H = X(XᵀX)⁻¹Xᵀ

Die erste Spalte von X ist ein 1-Vektor für den Achsenabschnitt β₀. H heißt Hat-Matrix, weil sie aus Y die "gehutzten" Werte Ŷ macht — ein orthogonaler Projektor auf den Spaltenraum von X. Für k=1 vereinfacht sich das zu den Formeln unten.

Die Kleinste-Quadrate-Idee (KQ / OLS)

Gesucht sind die Koeffizienten β̂₀, β̂₁, die die Summe der quadrierten Residuen minimieren — quadriert, damit sich positive und negative Abweichungen nicht gegenseitig aufheben und große Fehler stärker bestraft werden:

Formel: KQ-Schätzer (einfache Regression)
β̂₁ = Sxy / Sxx mit Sxy = Σ(xᵢ−x̄)(Yᵢ−Ȳ), Sxx = Σ(xᵢ−x̄)² β̂₀ = Ȳ − β̂₁·x̄

Bestimmtheitsmaß R²: wie viel erklärt das Modell?

Quadratsummenzerlegung
TSS = Σ(Yᵢ−Ȳ)² (Gesamtvariation) SSR = Σ(Ŷᵢ−Ȳ)² (durchs Modell erklärt) RSS = Σ(Yᵢ−Ŷᵢ)² (Rest, Residuen) TSS = SSR + RSS R² = SSR/TSS = 1 − RSS/TSS ∈ [0,1]

Gauß-Markov-Theorem (kurz)

BLUE — Best Linear Unbiased Estimator

Unter den Annahmen A1 (Vollrang von X), A2 (E[U]=0) und A3 (Cov(U)=σ²I) ist der KQ-Schätzer β̂ BLUE: der beste (kleinste Varianz) unter allen linearen, erwartungstreuen Schätzern. Normalverteilung (A4) ist dafür nicht nötig — die braucht man erst für t-Tests und Konfidenzintervalle der Koeffizienten.

AnnahmeNötig für
A1: Vollrang XKQ-Schätzer existiert eindeutig
A2: E[U] = 0Erwartungstreue von β̂
A3: Cov(U) = σ²IGauß-Markov (BLUE)
A4: U ~ Normalt-Tests, Konfidenzintervalle

Schritt-für-Schritt: Mini-Beispiel komplett durchgerechnet

Daten: (x,y) = (1,2), (2,3), (3,5). Gesucht: die Regressionsgerade ŷ = β̂₀ + β̂₁·x.

  1. Mittelwerte berechnen: x̄ = (1+2+3)/3 = 2,   ȳ = (2+3+5)/3 = 10/3 ≈ 3.333
  2. Sxx berechnen: Sxx = (1−2)²+(2−2)²+(3−2)² = 1+0+1 = 2
  3. Sxy berechnen:
    Sxy = (1−2)(2−10/3) + (2−2)(3−10/3) + (3−2)(5−10/3)
    = (−1)(−4/3) + 0 + (1)(5/3) = 4/3 + 5/3 = 9/3 = 3
  4. β̂₁ berechnen: β̂₁ = Sxy/Sxx = 3/2 = 1.5
  5. β̂₀ berechnen: β̂₀ = ȳ − β̂₁·x̄ = 10/3 − 1.5·2 = 10/3 − 3 = 1/3 ≈ 0.333
Ergebnis

Regressionsgerade: ŷ = 1/3 + 1.5·x. Interpretation von β̂₁: Steigt x um 1 Einheit, steigt der erwartete y-Wert im Schnitt um 1.5 Einheiten (ceteris paribus).

🧮 Klausurtyp: R² für dasselbe Beispiel berechnen

  1. Fitted values Ŷᵢ = 1/3 + 1.5·xᵢ berechnen:
    Ŷ(1) = 1/3+1.5 = 11/6 ≈ 1.833,   Ŷ(2) = 1/3+3 = 10/3 ≈ 3.333,   Ŷ(3) = 1/3+4.5 = 29/6 ≈ 4.833
  2. RSS berechnen: (2−1.833)² + (3−3.333)² + (5−4.833)² = (1/6)² + (1/3)² + (1/6)² = 1/36+4/36+1/36 = 6/36 = 1/6 ≈ 0.1667
  3. TSS berechnen: (2−10/3)² + (3−10/3)² + (5−10/3)² = 16/9+1/9+25/9 = 42/9 = 14/3 ≈ 4.667
  4. R² berechnen: R² = 1 − RSS/TSS = 1 − (1/6)/(14/3) = 1 − 1/28 = 27/28 ≈ 0.964
Einordnung

R² ≈ 0.964 bedeutet: 96.4% der Streuung in y wird durch die Gerade erklärt — bei nur 3 Datenpunkten ist das allerdings kein Wunder, mit so wenigen Punkten liegt R² fast automatisch hoch.

🧮 Klausurtyp: t-Test für einen Regressionskoeffizienten

Eine größere Studie (n=20, k=1) liefert β̂₁ = 0.82 mit Standardfehler SE(β̂₁) = 0.31. Test H₀: β₁=0 vs. H₁: β₁≠0 bei α = 0.05 — hat die Kovariable überhaupt einen Einfluss?

  1. Teststatistik: T = β̂₁/SE(β̂₁) = 0.82/0.31 ≈ 2.645
  2. Freiheitsgrade: df = n−k−1 = 20−1−1 = 18
  3. Kritischen Wert nachschlagen: t_{0.025, 18} ≈ 2.101
  4. Entscheidung: |T| = 2.645 > 2.101 → H₀ ablehnen — β₁ ist signifikant von 0 verschieden, die Kovariable hat einen nachweisbaren Einfluss

Typische Fallen

❌ Falschσ̂² = RSS/n
✅ Richtigσ̂² = RSS/(n−k−1). Bei einfacher Regression (k=1): RSS/(n−2) — zwei Freiheitsgrade gehen für β̂₀ und β̂₁ verloren.
❌ FalschR² = 0.8 ist grundsätzlich "schlecht".
✅ RichtigR² ist kontextabhängig: In den Sozialwissenschaften gilt 0.5 oft als gut, in der Physik erwartet man dagegen häufig 0.99+.
❌ FalschEin hohes R² beweist, dass alle Kovariablen wichtig sind.
✅ RichtigR² kann beim Hinzufügen weiterer Variablen nie sinken — auch nutzlose Variablen erhöhen R² leicht. Für Modellvergleiche nutzt man das korrigierte R̄² = 1 − (1−R²)·(n−1)/(n−k−1).
❌ FalschGauß-Markov (BLUE) gilt nur, wenn die Fehler normalverteilt sind.
✅ RichtigGauß-Markov gilt bereits unter A1–A3 — ganz ohne Normalverteilungsannahme. A4 (Normalverteilung) brauchst du erst für t-Tests und Konfidenzintervalle.
❌ Falschβ̂₁ > 0 bedeutet, x verursacht y.
✅ RichtigRegression zeigt Assoziation, keine Kausalität. Ein positiver Zusammenhang kann auch durch Drittvariablen oder Zufall entstehen.

Quick-Reference

🎯 Quick-Reference für die Klausur

Kernformeln

FormelBedeutung
β̂ = (XᵀX)⁻¹XᵀYKQ-Schätzer, allgemein (Matrixform)
β̂₁ = Sxy/SxxKQ-Steigung, einfache Regression
β̂₀ = Ȳ − β̂₁·x̄Achsenabschnitt, einfache Regression
σ̂² = RSS/(n−k−1)Varianzschätzung
R² = 1 − RSS/TSSBestimmtheitsmaß
T = β̂ⱼ/SE(β̂ⱼ) ~ t_{n−k−1}t-Test für einen Koeffizienten

Modellannahmen auf einen Blick

A1  Vollrang X        → KQ-Schätzer existiert eindeutig
A2  E[U] = 0           → Erwartungstreue
A3  Cov(U) = σ²I       → Gauß-Markov (BLUE)
A4  U ~ Normal         → t-Tests, Konfidenzintervalle

Merksätze

  • β̂₁ = Sxy/Sxx zuerst, dann β̂₀ = Ȳ − β̂₁·x̄ — immer in dieser Reihenfolge
  • TSS = SSR + RSS, R² = 1 − RSS/TSS
  • R² steigt fast automatisch mit mehr Variablen → R̄² für fairen Modellvergleich
  • Gauß-Markov braucht keine Normalverteilung — die kommt erst bei Tests/KI ins Spiel
  • Regression ≠ Kausalität

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.Daten (1,2),(2,3),(3,5): Wie groß ist Sxx = Σ(xᵢ−x̄)²?

x̄=2. (1−2)²+(2−2)²+(3−2)² = 1+0+1 = 2.

2.Gleiche Daten: Wie groß ist Sxy = Σ(xᵢ−x̄)(yᵢ−ȳ)?

ȳ=10/3. (−1)(−4/3)+0·(−1/3)+(1)(5/3) = 4/3+5/3 = 3.

3.Wie groß ist β̂₁ = Sxy/Sxx?

β̂₁ = 3/2 = 1.5.

4.🧮 Klausurtyp: Wie groß ist β̂₀ = ȳ − β̂₁·x̄?

β̂₀ = 10/3 − 1.5·2 = 10/3 − 3 = 1/3 ≈ 0.333.

5.Wie heißt das Verfahren, das β̂ durch Minimierung der quadrierten Residuen bestimmt?

Die Kleinste-Quadrate-Methode (KQ, englisch OLS).

6.Was bedeutet β̂₁ = 1.5 korrekt interpretiert?

Regression beschreibt einen Zusammenhang im Erwartungswert — keine deterministische und keine kausale Aussage.

7.Welche Annahmen braucht das Gauß-Markov-Theorem (β̂ ist BLUE)?

Gauß-Markov gilt bereits unter A1–A3. A4 (Normalverteilung) wird erst für t-Tests und Konfidenzintervalle der Koeffizienten benötigt.
