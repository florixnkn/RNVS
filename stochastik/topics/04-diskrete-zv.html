Worum geht's?

Eine Zufallsvariable X ist eigentlich nur ein Übersetzer: Sie nimmt ein Zufallsergebnis — das kann alles Mögliche sein, Symbole, Farben, Ereignisse — und übersetzt es in eine Zahl. Damit kannst du plötzlich rechnen, Mittelwerte bilden und Wahrscheinlichkeiten vergleichen, egal wie kompliziert das ursprüngliche Zufallsexperiment war.

Stell dir vor …

… du wirfst dreimal eine Münze. Das Ergebnis eines einzelnen Wurfs ist "Kopf" (K) oder "Zahl" (Z) — keine Zahl, mit der man vernünftig rechnen kann. Deine Wett-App interessiert aber nur eine Zahl: wie oft insgesamt Kopf kam. Die Zufallsvariable X = "Anzahl Kopf bei 3 Würfen" ist genau dieser Übersetzer: Sie schaut sich das rohe Ergebnis ω ∈ Ω an (z. B. ω = KZK) und gibt eine Zahl zurück (X(KZK) = 2). Ω bleibt das komplizierte Ur-Ergebnis, X macht daraus etwas, mit dem du arbeiten kannst.

X als Übersetzer, Träger & Wahrscheinlichkeitsfunktion

Formal ist eine Zufallsvariable eine Funktion X: Ω → ℝ, die jedem Ergebnis des Zufallsexperiments eine reelle Zahl zuordnet. Sie heißt diskret, wenn sie nur abzählbar viele Werte annehmen kann (z. B. {0,1,2,3} oder ganz ℕ) — im Gegensatz zu stetigen Zufallsvariablen, die jeden Wert in einem Intervall annehmen können (dazu mehr in Thema 06).

Träger: wo X überhaupt landen kann

Formel: Träger
T_X = supp(X) = { x ∈ ℝ : P(X = x) > 0 }

Der Träger ist die Menge aller Werte, die X mit positiver Wahrscheinlichkeit annimmt. Bei unserem Münz-Beispiel: T_X = {0, 1, 2, 3}. Bei einem Würfelwurf: T_X = {1,2,3,4,5,6}. Bei "Anzahl Würfe bis zur ersten 6": T_X = {1, 2, 3, …} = ℕ, also unendlich, aber immer noch abzählbar.

Wahrscheinlichkeitsfunktion p(x)

Die Wahrscheinlichkeitsfunktion (kurz WK-Funktion, meist f_X oder p_X genannt) ordnet jedem Wert im Träger seine Wahrscheinlichkeit zu. Für unser Münz-Beispiel — 8 gleichwahrscheinliche Ausgänge, davon 1× "3 Kopf", 3× "2 Kopf", 3× "1 Kopf", 1× "0 Kopf":

Formel: Wahrscheinlichkeitsfunktion
f_X(x) = P(X = x)   ·   f_X(x) ≥ 0   ·   Σ_{x ∈ T_X} f_X(x) = 1

Zwei Eigenschaften musst du bei jeder WK-Funktion checken: nirgendwo negativ, und die Summe über den ganzen Träger ist genau 1. Das ist deine eingebaute Fehlerprobe.

Verteilungsfunktion F(x): die Treppe

Die Verteilungsfunktion sammelt die Wahrscheinlichkeiten von "links" auf: F_X(x) = P(X ≤ x). Bei einer diskreten ZV ist das immer eine Treppenfunktion — konstant zwischen den Trägerpunkten, mit Sprüngen genau dort, wo X einen Wert annehmen kann. Die Sprunghöhe an einer Stelle x entspricht exakt f_X(x).

Formel: Verteilungsfunktion
F_X(x) = P(X ≤ x) = Σ_{t ≤ x, t ∈ T_X} f_X(t)

Eigenschaften: monoton nicht-fallend, rechtsseitig stetig, F_X(x) → 0 für x → −∞ und F_X(x) → 1 für x → +∞.

P(a < X ≤ b)= F(b) − F(a)
P(a ≤ X ≤ b)= F(b) − F(a) + f(a)
P(X > a)= 1 − F(a)
P(X = a)= F(a) − F(a⁻)

Schritt-für-Schritt: Verteilungsfunktion aufstellen & ablesen

Nehmen wir eine ZV X mit Träger {1, 2, 3} und P(X=1) = 0.2, P(X=2) = 0.5, P(X=3) = 0.3.

  1. Probe machen: 0.2 + 0.5 + 0.3 = 1.0 ✓ — passt.
  2. Trägerpunkte sortieren: 1 < 2 < 3.
  3. F_X als Stufenfunktion aufschreiben:
    F_X(x) = 0 für x < 1
    F_X(x) = 0.2 für 1 ≤ x < 2
    F_X(x) = 0.7 für 2 ≤ x < 3
    F_X(x) = 1.0 für x ≥ 3
  4. Probe: letzte Stufe = 1 ✓, alle Sprünge positiv (0.2, 0.5, 0.3) ✓

Wahrscheinlichkeiten aus F_X ablesen

Die größte Falle: ≤ vs. <

F_X(x) = P(X ≤ x) — die Grenze ist immer inklusive. Für P(X < 2) darfst du nicht F_X(2) nehmen, sondern musst zur vorherigen Trägerstufe: F_X(1).

  1. P(X ≤ 2) = F_X(2) = 0.7
  2. P(X < 2) = F_X(1) = 0.2 (nicht F_X(2)!)
  3. P(X = 2) = F_X(2) − F_X(1) = 0.7 − 0.2 = 0.5
  4. P(1 < X ≤ 3) = F_X(3) − F_X(1) = 1.0 − 0.2 = 0.8

Zweites Beispiel: die Inklusive-Untergrenze-Falle

X hat Träger {0,1,2,3} mit p(0)=0.1, p(1)=0.3, p(2)=0.4, p(3)=0.2. Gesucht: P(1 ≤ X ≤ 3).

  1. F_X aufstellen: F(0)=0.1, F(1)=0.4, F(2)=0.8, F(3)=1.0
  2. Naiv (falsch): F(3) − F(1) = 1.0 − 0.4 = 0.6 — das ist P(1 < X ≤ 3), nicht P(1 ≤ X ≤ 3)!
  3. Richtig: P(1 ≤ X ≤ 3) = F(3) − F(1) + f(1) = 0.6 + 0.3 = 0.9
  4. Gegenprobe direkt über die WK-Funktion: p(1)+p(2)+p(3) = 0.3+0.4+0.2 = 0.9 ✓

Typische Fallen

❌ FalschP(X < 2) = F_X(2), weil "kleiner 2 ist doch fast dasselbe wie kleiner gleich 2".
✅ RichtigP(X < 2) = F_X(1) bei einer diskreten ZV mit Trägerpunkten in ℕ — du musst zur vorherigen Trägerstufe springen, nicht F_X(2) nehmen.
❌ FalschMan vergisst, alle Trägerpunkte zu summieren, und Σf_X(x) kommt nicht auf 1 — aber das fällt nicht auf.
✅ RichtigΣ_{x∈T_X} f_X(x) = 1 muss immer gelten — das ist deine Pflicht-Probe nach jeder Aufgabe.
❌ FalschF_X verläuft zwischen den Trägerpunkten schräg oder gekrümmt, so wie eine "normale" Funktion.
✅ RichtigF_X ist bei diskreten ZV eine Treppenfunktion: konstant zwischen den Trägerpunkten, Sprung nur genau an ihnen.
❌ FalschP(1 ≤ X ≤ 3) = F_X(3) − F_X(1)
✅ RichtigP(1 ≤ X ≤ 3) = F_X(3) − F_X(1) + f_X(1) — die Untergrenze zählt mit, weil ≤ nicht < heißt.

Quick-Reference

🎯 Quick-Reference für die Klausur

Kernformeln

FormelBedeutung
f_X(x) = P(X = x)Wahrscheinlichkeitsfunktion
Σ f_X(x) = 1Normierung über den ganzen Träger
F_X(x) = P(X ≤ x)Verteilungsfunktion (Treppenfunktion)
P(a < X ≤ b) = F(b) − F(a)WK aus VF
P(a ≤ X ≤ b) = F(b) − F(a) + f(a)inklusive Untergrenze
P(X = a) = F(a) − F(a⁻)Sprunghöhe an der Stelle a

Merksätze

  • ≤ ist immer inklusive — F_X(x) = P(X ≤ x), nicht P(X < x)
  • F_X ist eine Treppenfunktion bei diskreten ZV — Sprunghöhe = f_X an der Stelle
  • Erst Trägerpunkte sortieren, dann F_X stufenweise aufschreiben
  • Immer Σf_X(x)=1 prüfen — die günstigste Fehlerkontrolle überhaupt
  • Bei P(a ≤ X ≤ b): Untergrenze extra addieren (f(a) dazu)

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.Wie heißt die Menge {x ∈ ℝ : P(X=x) > 0}, also alle Werte, die X mit positiver Wahrscheinlichkeit annimmt?

Das ist der Träger (support) von X, geschrieben T_X oder supp(X).

2.X hat Träger {1,2,3} mit P(X=1)=0.2, P(X=2)=0.5, P(X=3)=0.3. Wie groß ist F_X(2) = P(X ≤ 2)?

F_X(2) = P(X=1)+P(X=2) = 0.2+0.5 = 0.7.

3.Gleiches Beispiel: Wie groß ist P(X < 2)? Achtung, Falle!

P(X<2) = F_X(1) = 0.2 — nicht F_X(2)! Die Grenze bei F ist inklusive (≤), also musst du zur vorherigen Stufe.

4.Gleiches Beispiel: Wie groß ist P(1 < X ≤ 3)?

F_X(3) − F_X(1) = 1.0 − 0.2 = 0.8.

5.F_X ist bei einer diskreten Zufallsvariable vom Funktionstyp her eine …

F_X springt nur an Trägerpunkten und ist dazwischen konstant — eine klassische Treppenfunktion.

6.Welche Aussage über eine diskrete Zufallsvariable ist korrekt?

Die Sprunghöhe von F_X an der Stelle a ist genau f_X(a) = P(X=a). F_X ist monoton nicht-fallend (nicht fallend), und f_X(x) > 1 gibt es bei diskreten ZV nicht — das ist nur bei Dichten stetiger ZV möglich (Thema 06).

7.🧮 Klausurtyp: X hat Träger {0,1,2,3} mit p(0)=0.1, p(1)=0.3, p(2)=0.4, p(3)=0.2. Wie groß ist P(1 ≤ X ≤ 3)?

F(3) − F(1) + f(1) = 1.0 − 0.4 + 0.3 = 0.9. Gegenprobe: p(1)+p(2)+p(3) = 0.3+0.4+0.2 = 0.9 ✓.
