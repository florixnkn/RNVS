Worum geht's?

Fast jede Klausuraufgabe zu Zufallsvariablen läuft auf dieselbe erste Frage hinaus: welche der fünf Standard-Verteilungen passt hier? Erkennst du das Muster richtig, ist der Rest nur noch Formel einsetzen. Erkennst du es falsch, rechnest du mit der komplett falschen Formel weiter — deshalb ist dieses Thema eines der klausurrelevantesten im ganzen Modul.

Stell dir vor …

… du hast einen Werkzeugkasten mit fünf Werkzeugen, und jedes passt nur zu einer bestimmten Art von Schraube. Ein einzelner Handgriff (ein Versuch, ja/nein) → Bernoulli. Mehrmals dasselbe unabhängig wiederholen und Erfolge zählen → Binomial. Weiterarbeiten, bis es endlich klappt → Geometrisch. Aus einer begrenzten Kiste ohne Zurücklegen ziehen → Hypergeometrisch. Seltene Ereignisse über die Zeit zählen (E-Mails pro Stunde) → Poisson. Die Klausur testet vor allem, ob du das richtige Werkzeug aus dem Kasten nimmst.

Die fünf diskreten Standard-Verteilungen

Bernoulli Ber(p) — der einzelne Versuch

Der Grundbaustein von allem: ein einziger Versuch mit Erfolg (1) oder Misserfolg (0). Münzwurf, ein übertragenes Bit, ein Kunde kauft oder kauft nicht.

Formel: Bernoulli
P(X=1) = p  ·  P(X=0) = 1−p =: q  ·  E[X] = p  ·  Var(X) = p(1−p)

Geometrisch Geom(p) — bis zum ersten Erfolg

Zählt die Anzahl Versuche bis (und mit) dem ersten Erfolg — Träger startet bei 1, nicht bei 0! Eine Besonderheit: die geometrische Verteilung ist gedächtnislos — egal wie oft du schon erfolglos warst, die Restwahrscheinlichkeit sieht immer gleich aus: P(X>m+n | X>m) = P(X>n).

Formel: Geometrisch
P(X=k) = (1−p)k−1·p  für k=1,2,3,…  ·  E[X] = 1/p  ·  Var(X) = (1−p)/p²

Binomial B(n,p) — n unabhängige Bernoulli-Versuche

Die Summe von n unabhängigen Ber(p)-Versuchen: X = X₁+…+Xₙ. Zählt, wie viele von n Versuchen erfolgreich waren. Spezialfall B(1,p) = Ber(p).

Formel: Binomial
P(X=k) = C(n,k)·pk·(1−p)n−k  ·  E[X] = np  ·  Var(X) = np(1−p)

Hypergeometrisch H(N,M,n) — ziehen ohne Zurücklegen

N = Größe der Gesamtpopulation, M = Anzahl "Erfolgs"-Elemente darin, n = Stichprobengröße. Weil ohne Zurücklegen gezogen wird, hängen die einzelnen Ziehungen voneinander ab — deshalb ist die Formel komplizierter als bei Binomial. Ist N sehr viel größer als n, macht es kaum noch einen Unterschied, ob zurückgelegt wird oder nicht: H(N,M,n) ≈ B(n, M/N).

Formel: Hypergeometrisch
P(X=k) = C(M,k)·C(N−M,n−k) / C(N,n)  ·  E[X] = n·M/N

Poisson Poi(λ) — seltene Ereignisse zählen

λ ist die erwartete Anzahl Ereignisse im betrachteten Intervall (Zeit, Fläche, …). Poisson entsteht als Grenzfall der Binomialverteilung, wenn n sehr groß und p sehr klein wird, aber n·p ≈ λ konstant bleibt — genau das Muster von "seltenen" Ereignissen bei "vielen" Gelegenheiten.

Formel: Poisson
P(X=k) = e−λ·λk/k!  ·  E[X] = λ  ·  Var(X) = λ

Einzigartig unter allen Standardverteilungen: E[X] = Var(X) = λ. Das ist ein starkes Erkennungsmerkmal — wenn in einer Aufgabe E[X] ≠ Var(X) herauskommt, ist es keine Poisson-Verteilung.

Durchgerechnete Beispiele

Binomial: X ~ B(10, 0.3). Berechne P(X=2).

  1. Binomialkoeffizient: C(10,2) = 10!/(2!·8!) = 45
  2. p-Potenz: 0.3² = 0.09
  3. (1−p)-Potenz: 0.7⁸ ≈ 0.057648
  4. Alles multiplizieren: P(X=2) = 45 · 0.09 · 0.057648 ≈ 0.2335

Poisson: X ~ Poi(2). Berechne P(X=0).

  1. Formel einsetzen: P(X=0) = e⁻²·2⁰/0! = e⁻²·1/1
  2. e⁻² auswerten: e⁻² ≈ 0.13534
  3. Ergebnis: P(X=0) ≈ 0.1353

Hypergeometrisch: 20 Kugeln, 8 rot, 12 blau. Ziehe 5 ohne Zurücklegen. P(genau 2 rot)?

  1. Parameter identifizieren: N=20, M=8 (rot), n=5, k=2
  2. Zähler: C(8,2)·C(12,3) = 28 · 220 = 6160
  3. Nenner: C(20,5) = 15504
  4. Ergebnis: P(X=2) = 6160/15504 ≈ 0.3974

Weiteres Beispiel: Binomial mit kumulierter Wahrscheinlichkeit

10 Pakete werden verschickt, jedes unabhängig mit P=0.16 verspätet. X = Anzahl verspäteter Pakete, X ~ B(10, 0.16). Gesucht: P(X ≤ 1), sowie E[X] und Var(X).

  1. P(X=0): 0.84¹⁰ ≈ 0.1749
  2. P(X=1): C(10,1)·0.16·0.84⁹ = 10·0.16·0.2082 ≈ 0.3331
  3. P(X ≤ 1) = P(X=0) + P(X=1): 0.1749 + 0.3331 ≈ 0.508
  4. E[X] = np: 10·0.16 = 1.6
  5. Var(X) = np(1−p): 10·0.16·0.84 = 1.344

Typische Fallen

❌ Falsch"5 Karten aus 52 ziehen" → einfach Binomial verwenden, weil es die bekannteste Formel ist.
✅ RichtigOhne Zurücklegen aus einer endlichen Population → Hypergeometrisch. Nur wenn N ≫ n gilt, ist die Binomial-Näherung akzeptabel.
❌ FalschBei Geom(p) ist P(X=0) = (1−p)⁰·p = p der erste Term.
✅ RichtigDer Träger startet bei 1, nicht bei 0 — der erste Term ist P(X=1) = p. (Es gibt eine alternative Variante mit Träger {0,1,…} für die Anzahl Misserfolge — im Zweifel steht dabei, welche gemeint ist.)
❌ FalschP(X=3) = p³·(1−p)^{n−3} — den Binomialkoeffizienten vergisst man leicht.
✅ RichtigP(X=3) = C(n,3)·p³·(1−p)^{n−3} — der Koeffizient zählt, auf wie viele Arten die 3 Erfolge unter n Versuchen verteilt sein können.
❌ FalschE[X] = Var(X) gilt ungefähr bei jeder Verteilung mit kleinen Werten.
✅ RichtigDiese Gleichheit ist einzigartig für Poisson. Kommt in einer Aufgabe E[X] ≠ Var(X) heraus, ist Poisson die falsche Wahl.
❌ FalschDer Träger der Hypergeometrischen ist immer {0, 1, …, n}, genau wie bei Binomial.
✅ RichtigUntere Grenze ist max(0, n+M−N) — wenn n+M > N, müssen zwangsläufig mindestens n+M−N Erfolge gezogen werden, weil sonst nicht genug "Nicht-Erfolge" übrig wären.

Entscheidungshilfe

Ohne Zurücklegen, endliche Population?    → Hypergeometrisch H(N,M,n)
Nur EIN Versuch?                          → Bernoulli Ber(p)
Feste Anzahl Versuche, zähle Erfolge?     → Binomial B(n,p)
Bis zum ERSTEN Erfolg?                    → Geometrisch Geom(p)
Seltenes Ereignis / Intensität?           → Poisson Poi(λ)

Merksätze

  • Erste Frage immer: mit oder ohne Zurücklegen?
  • Poisson: E[X] = Var(X) = λ — einzigartiges Erkennungsmerkmal
  • Geometrisch startet bei k=1, nicht bei k=0
  • Binomialkoeffizient C(n,k) nie vergessen bei Binomial und Hypergeometrisch
  • Approximationen: H(N,M,n) ≈ B(n,M/N) für N≫n  ·  B(n,p) ≈ Poi(np) für n groß, p klein

🧩 Wissenscheck

Teste dich selbst! Tippe deine Antwort ein oder wähle eine Option — du bekommst sofort Rückmeldung. Mit „Lösung" siehst du die Erklärung.

1.Du ziehst 5 Karten aus einem 52er-Kartendeck OHNE Zurücklegen und zählst die Asse. Welche Verteilung?

Ohne Zurücklegen aus einer endlichen Population (52 Karten) → Hypergeometrisch H(52, 4, 5).

2.Welche Verteilung hat Träger {1,2,3,…} und beschreibt die Anzahl Versuche bis zum ERSTEN Erfolg?

Die geometrische Verteilung Geom(p) — zählt die Versuche bis (und mit) dem ersten Erfolg.

3.X ~ B(10, 0.3). Wie groß ist E[X]?

E[X] = n·p = 10·0.3 = 3.

4.X ~ B(10, 0.3). Wie groß ist P(X=2)?

C(10,2)·0.3²·0.7⁸ = 45 · 0.09 · 0.05765 ≈ 0.2335.

5.X ~ Poi(2). Wie groß ist P(X=0)?

P(X=0) = e⁻²·2⁰/0! = e⁻² ≈ 0.1353.

6.Bei welcher Verteilung gilt IMMER E[X] = Var(X)?

Nur bei der Poisson-Verteilung gilt immer E[X] = Var(X) = λ — bei den anderen vier sind Erwartungswert und Varianz strukturell verschieden.

7.🧮 Klausurtyp: 20 Kugeln, 8 rot, 12 blau. Du ziehst 5 ohne Zurücklegen. Wie groß ist P(genau 2 rot)?

P(X=2) = C(8,2)·C(12,3)/C(20,5) = 28·220/15504 = 6160/15504 ≈ 0.3974.
